# Mutating an array

The spread operator is enough for working with objects.

What if you want to remove or replace something from an array?
Then you need to use a corresponding *array method*.

## Array Methods for Mutations

| Operation | Array method |
|-----------|--------------|
| Add an element | spread |
| Remove an element | filter, slice |
| Replace an element | map |
