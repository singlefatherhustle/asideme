# A simple example

We can also think of `sum(n)` in relation to the solution of the previous problem.

```
sum(1) = 1
sum(2) = 1 + 2                = sum(1) + 2 = 3
sum(3) = 1 + 2 + 3            = sum(2) + 3 = 6
sum(4) = 1 + 2 + 3 + 4        = sum(3) + 4 = 10
```
