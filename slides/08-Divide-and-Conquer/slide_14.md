# Merge sort time complexity

Given an array of size n:

- How many times can we split the array in half?
  - log(n)
- How do we merge two halves?
  - By looking at every element in both halves
  - O(n)

The time complexity of merge sort is O(log(n) * n) = O(nlog(n)).
