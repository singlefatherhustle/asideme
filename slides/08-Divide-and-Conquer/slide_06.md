# A simple example

Given an integer n, the function `sum` returns the sum of all non-negative integers up to n.

```
sum(1) = 1
sum(2) = 1 + 2                = 3
sum(3) = 1 + 2 + 3            = 6
sum(4) = 1 + 2 + 3 + 4        = 10
```

## Visual Representation

The following diagrams show the pattern for n = 1, 2, 3, and 4:

**n = 1:** 1 square

**n = 2:** 3 squares arranged in a staircase pattern (1 on top, 2 below)

**n = 3:** 6 squares arranged in a staircase pattern (1 on top, 2 in middle row, 3 on bottom)

**n = 4:** 10 squares arranged in a staircase pattern (1 on top, 2 in second row, 3 in third row, 4 on bottom)
