# Add or mutate a property in an object

What if we want to add a new property, or change the value of an existing property? We use the assignment operator (=) for both!

```javascript
// This creates a new property if the key doesn't already exist
weather.temperature = 24.1;
```

```javascript
// This mutates the value if the key does already exist
weather.temperature = 72.6;
```
