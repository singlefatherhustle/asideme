# Access a value with dot notation

To access a value in an object, we need to refer to the corresponding key. We use **dot notation** if we know the name of the key.

```js
const weather = {
  temperature: 29.4,
  forecast: "partly cloudy",
  humidity: 0.38,
  precipitation: 0.16,
};
```

```js
weather.temperature;  // 29.4
weather.forecast;     // "partly cloudy"
weather.wind;         // undefined
```
