# Objects

An **object** is a collection of **properties** denoted by curly braces.
A property is a **key:value** pair.

```js
const weather = {
    temperature: 29.4,
    forecast: "partly cloudy",
    humidity: 0.38,
    precipitation: 0.16,
};
```
