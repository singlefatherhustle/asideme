# Recap

## Key Concepts

- An array holds a **group** of multiple similar items.
- An **object** holds a **single item** with multiple properties.
- We use **dot notation** to access an object's value with a static key.
- We use **bracket notation** to access an object's value with a dynamic key.
- We can interact with objects by adding, removing, or modifying key-value pairs.
- We iterate through an object's properties with the **for...in** loop.
