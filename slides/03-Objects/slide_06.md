# Shape of an object

An object's **shape** refers to the names of its keys and the types of their values. What is the shape of **weather**?

## Code Example

```js
const weather = {
  temperature: 29.4,
  forecast: "partly cloudy",
  humidity: 0.38,
  precipitation: 0.16,
};
```

## Shape Definition

```
{
  temperature: number;
  forecast: string;
  humidity: number;
  precipitation: number;
}
```
