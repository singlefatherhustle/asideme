# What about space complexity?

We also use big O notation to analyze **space complexity**!

We measure the amount of memory required by the algorithm based on the size of the input.

Very roughly speaking, how many values need to be stored?

- 1 variable = 1 value
- An array of n elements = n values

For example, the space complexity of copying an array of n elements is O(n).
