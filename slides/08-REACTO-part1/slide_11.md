# Big O comparisons

These are the most common big Os, starting with the fastest (slowest growing) and ending with the slowest (fastest growing).

## Big O Notation Table

| Big O | Common Name |
|-------|-------------|
| O(1) | constant |
| O(log n) | logarithmic |
| O(n) | linear |
| O(n²) | quadratic |
| O(n³) | cubic |
| O(2ⁿ) | exponential |
| O(n!) | factorial |
