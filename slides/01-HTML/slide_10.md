# Prefer semantic elements

## Comparison: `<h1>` vs `<span style="font-size: 32px; margin: 21px 0;">`

### `<h1>`

- Indicates the importance of the content
- Allows for quick navigation between different headings of the same level
- Provides a label for following content
- Is quickly read by developers as "Heading Level 1"

### `<span ...>`

This might render the same to some browsers, but...

- it has no semantic value
- provides no meaning to browsers or screen readers
- is more difficult for developers to read and skim
