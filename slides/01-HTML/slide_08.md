# Attributes

Tags can also contain additional information called **attributes**.

An attribute can indicate where an element will link to when clicked, the type of files an upload button will allow, alternative text in case an image can't be displayed, etc.

Attributes are written in the *opening tag* and separated by spaces. Each attribute must have a **name** and often a corresponding **value**.

## Attribute Syntax Diagram

```html
<input type="number" required>
```

| Component | Content |
|-----------|---------|
| `<` | Opening bracket |
| tag name | `input` |
| `_` | Space |
| attribute name | `type` |
| `=` | Equals sign |
| attribute value | `"number"` |
| `_` | Space |
| attribute name | `required` |
| `>` | Closing bracket |
