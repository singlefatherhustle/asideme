# HTML

## What is HTML?

**HTML** stands for **Hyper Text Markup Language**.

## Let's break it down:

### Hyper Text
metadata: text that provides information about some other text

### Markup Language
a language that specifies the structure, formatting, and content of a document

## Putting it back together...

HTML is a language where we write content, and then we write text to specify metadata about that content, such as structure or formatting.
