# HTML

Let's take a look at our very first element in HTML.
The *content* of the element is the text "Weather Forecast".

What is *h1*? It is *metadata* about that content.
In this case, *h1* tells us that this text is a heading, and it ranks #1 in importance.

## Code Example

```html
<h1>Weather Forecast</h1>
```

## HTML Element Structure Diagram

| metadata | content | metadata |
|----------|---------|----------|
| `<h1>` | Weather Forecast | `</h1>` |
