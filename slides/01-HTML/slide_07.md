# Tags

In HTML, this metadata is called a **tag** because the content is **tagged** with this information, like a price tag!

Tags are how we provide information such as importance (h1 vs h2), media type (audio vs video), or content type (blockquote vs table).

## HTML Tag Structure

```html
<h1>Weather Forecast</h1>
```

### Diagram: Tag Components

| Component | Description |
|-----------|-------------|
| opening tag | `<h1>` |
| content | `Weather Forecast` |
| closing tag | `</h1>` |
| **element** | The complete combination of opening tag + content + closing tag |
