# HTML

So how do you create a document? By writing HTML!
A **browser** is a program that can **parse** (read) an HTML file to **render** (display) a document.

## Process Diagram

```
index.html → browser → document
```

**Diagram Structure:**
- Left node: Document icon labeled "index.html"
- Center node: Globe icon labeled "browser"
- Right node: Browser window icon labeled "document"
- Arrows connecting left → center → right showing the flow of processing
