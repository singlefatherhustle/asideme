# R = Read and Repeat the Problem

~5 minutes

Write a function that, given two sorted arrays, finds the number of elements in common.

Read the problem out loud and ask clarifying questions to both make sure that you understand the problem, and to demonstrate to the interviewer that you understand it.

- What types are are the array elements? numbers, strings, booleans?
- Are the array elements the same type?
- Are the arrays sorted in ascending or descending order?
- Can the arrays be empty?
- Can the arrays contain duplicates, or are the elements unique?
- Do I want the actual elements in common, or just the count?
- Are we guaranteed that there will be elements in common?
