# What is a technical interview?

A traditional **technical interview** involves writing code to solve an algorithm problem. The interviewer will assess your performance based on a rubric like the following:

- **technical knowledge**
  - did you illustrate several solutions and choose the optimal data structures and/or algorithms?

- **coding ability**
  - did you write clean and working code with no syntax errors?
  - did you demonstrate an understanding of the language paradigms?

- **communication**
  - did you clearly articulate your thought process?
  - are your answers well-organized and succinct?
