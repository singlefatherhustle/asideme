# Modify a selected element

Once an element is selected, you can interact with it like any other JavaScript object by reading and writing its properties.

```js
const blurb = document.querySelector("p.blurb");

console.log(blurb.textContent);
blurb.style.backgroundColor = "black";
blurb.style.color = "white";
```
