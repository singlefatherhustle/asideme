# Document Object Model

The **Document Object Model (DOM)** bridges the gap between HTML and JavaScript. It allows us to *model* the *document* as *objects*.

The DOM is a programming **interface** that connects two parts: the browser's render engine and the browser's JavaScript engine.

## DOM Flow Diagram

```
index.js → JS engine → DOM → render engine → document
```

**Component Details:**
- **index.js** - JavaScript source file
- **JS engine** - Processes JavaScript code (orange gear icon)
- **DOM** - Document Object Model bridge (blue spherical icon)
- **render engine** - Renders to the browser (orange gear icon)
- **document** - Final rendered output in browser window
