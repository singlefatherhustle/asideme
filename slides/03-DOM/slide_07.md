# The document object

We access the DOM from our JavaScript by interacting with the **document** object, which is an object with properties and methods that represent the document.

For example:

- **document.links** returns all `<a>` elements in a document
- **document.URL** returns the document location as a string
- **document.title** gets or sets the current title of the document
