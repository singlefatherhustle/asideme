# What is a table?

A **table** is composed of **columns** and **rows**, like a spreadsheet.

- Each **row** (a.k.a. **record**) represents an individual instance of the data.
- Each **column** (a.k.a. property, field, attribute) defines the type of information that is stored about each record.
