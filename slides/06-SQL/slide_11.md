# Database management systems

A **database management system (DBMS)** is software that helps us organize and interact with database files. It is able to directly manipulate the binary data in a database.

There are many DBMS options. We will be using **PostgreSQL**, which is free and open-source.

## Diagram

```
PostgreSQL ←—— binary ——→ database
```
