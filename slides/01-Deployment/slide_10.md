# Presenting… someone else's computer!

There's a solution that achieves everything we're looking for: renting someone else's computer! By **deploying** our project to a remote computer, we now have a way to share our website with the internet *without* any risk to our own machine.

## Deployment Architecture Diagram

```
your computer → index.html → rented computer ← internet ← [multiple client computers]
                                    ↑
                                    └─ connections from clients on internet
```

### Components:
- **your computer** — source of the project
- **index.html** — the website file being deployed
- **rented computer** — remote server hosting the website
- **internet** — the network connecting everything
- **[client computers]** — multiple users accessing the website from the internet
