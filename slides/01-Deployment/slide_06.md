# How can someone else access your website?

What if you wanted to share your website with someone else?
They don't have access to your project files.

## Diagram: Website Access Flow

```
someone else's computer → [X] → index.html → browser
                                    ↑
                            your computer
```

**Components:**
- **someone else's computer** (left): Remote user attempting to access the website
- **[X]** (red X symbol): Indicates no direct access to project files
- **index.html** (center box): The website file
- **browser** (right, globe icon): Where the website is displayed
- **your computer** (bottom): The origin/source of the website files
