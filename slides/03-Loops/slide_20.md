# Step through code

When the debugger is stopped at a breakpoint, you can step through it using the buttons in the toolbar. For now, you can focus on these two buttons.

- **play** = run code to the next breakpoint
- **step over** = advance to the next line of code

## Debugging Toolbar Buttons

The following buttons are available in the debugger toolbar:

| Button | Label | Keyboard Shortcut |
|--------|-------|-------------------|
| ⏸ | Play/pause | F8 |
| ↻ | Step over | F10 |
| ↗ | Step in | F11 |
| ↗ | Step out | Shift + F11 |
