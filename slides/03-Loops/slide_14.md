# Syntactic sugar

**Syntactic sugar** refers to *syntax* that allows us to write some code in a shorter and "sweeter" form.

For example, the following three lines of code all do the same thing: assign the *value* `i+1` to the *variable* `i`.

```
i++;
// is syntactic sugar for
i += 1;
// is syntactic sugar for
i = i + 1;
```
