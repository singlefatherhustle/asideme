# While loops

A while loop will repeat its **body of code** as *long as* its **condition** evaluates to true.

```
while (condition){
    // code A
}

// Some other code B
```

## Flowchart

The diagram shows the control flow of a while loop:

- **condition** (diamond shape at top) — evaluates the loop condition
- Arrow loops back from the condition to itself (feedback loop)
- **true** path (downward arrow) — leads to code block A (green circle)
- From code block A, arrow loops back up to the condition
- **false** path (leftward arrow) — exits the loop and proceeds to code block B (orange circle)
