# Recap

- A **loop** is a control structure that repeats code.
- An **infinite** loop is a loop that repeats code *forever*.
- An **indefinite** loop repeats code an *uncertain* number of times.
- A **definite** loop repeats code a *certain* number of times.
- A **for loop** is **syntactic sugar** for a **while loop**.
