# Use declarative mode

The JavaScript ecosystem is in constant flux, and libraries are always changing and updating. React Router has gone through multiple major versions, each of which drastically changes what the code looks like.

We are using the latest React Router v7 in declarative mode.

If/when you use external sources, make sure that you are looking at code for the correct version of React Router.
