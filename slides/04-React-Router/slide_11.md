# What does the code look like?

The Guided Practice will help you figure out the code on your own!

The ability to read documentation is one of **the most important skills** a software developer should have.

Very few web applications will use the **exact same tech stack**. Many teams even build their own in-house libraries that are not publicly available. That's how React started!

As a junior developer, you will be expected to know how to

- read documentation for an unfamiliar library
- ask clear and specific questions about the code
