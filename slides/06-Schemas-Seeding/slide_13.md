# Seed a database

When working with a database, it is a common practice to populate the database with some initial data. This is called **seeding** the database.

- Test functionality with sample data
- Ensure a consistent set of data for developers to work with
- Provide predefined values
  - geographic data
  - configuration defaults
  - product categories
