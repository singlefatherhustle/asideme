# Relative units

Instead of using absolute units, **prefer** to use **relative units** instead! These are *relative to* some other measurement on the site (usually font or viewport size).

## vw, vh = viewport width / height

```css
width: 20vw = set width to 20% of the viewport width
```

## % = percentage of the parent

```css
height: 75%; = set height to 75% of the parent element's height
```
