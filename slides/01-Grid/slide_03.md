# Responsive Design

A significant portion of all internet traffic is through mobile devices! That's why we design websites to be **responsive** to different screen / viewport sizes.

**Mobile-first design** means designing the mobile version of a website *first*.

- A mobile design might not look amazing, but it is still *functional* on a desktop.
- A desktop design will completely *break* on a mobile device!
