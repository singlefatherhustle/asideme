# Justify-content

We can move the flex items by applying different declarations to the flex **container**. **justify-content** distributes the free space on the **main axis**.

## Diagram: Flex Container with justify-content Distribution

```
flex container
┌─────────────────────────────────────────────────────────┐
│                                                           │
│  free        ┌───┐  ┌───┐  ┌───┐      free             │
│  space   ←→  │ A │  │ B │  │ C │  ←→  space    main axis
│             └───┘  └───┘  └───┘                        │
│                                                           │
└─────────────────────────────────────────────────────────┘
```
