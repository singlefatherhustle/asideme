# Align-items

`align-items` will move items along the *cross axis*.

## Diagram

```
align-items: start                          cross
                                            axis
    ┌─────────────────────────────┐         ↑
    │ A           │               │         │
    ├─────────────┼───────────────┤         │
    │             │       C       │ center  │
    │             ├───────────────┤         │
    │      B      │               │         │
    └─────────────────────────────┘         ↓
           end
```
