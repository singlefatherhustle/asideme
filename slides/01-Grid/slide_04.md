# Absolute units

How do you achieve responsive design? One method is by using the correct **units**.

An **absolute unit** is generally considered to always be the same size. These are more useful for *print*, rather than screen output.

## Examples:
`in`, `cm`, `mm`, `pt`, `px`

**Avoid** using absolute units if possible!
By definition, they *do not* respond to different screen sizes.
