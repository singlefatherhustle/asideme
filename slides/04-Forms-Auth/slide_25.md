# What is a token?

There are different types of tokens. We will be using **JSON Web Tokens (JWT)**.

A JWT is a very long obscured string with some encoded JSON data. This data is called the token's **payload**.

```

```

We will be spending more time with JWT in our future backend unit. For now, we can just trust an API to send this string back in its responses.
