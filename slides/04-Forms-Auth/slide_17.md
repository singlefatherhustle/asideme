# Send credentials

Upon the submission of a login form, a client sends a request to the API with some **credentials**, such as a username and a password.

## Diagram

```
Client ──[POST /login credentials]──> API
```

### Components:
- **Client**: Left figure (stick person)
- **POST /login credentials**: Center black box with login icon
- **API**: Right figure (stick person)
