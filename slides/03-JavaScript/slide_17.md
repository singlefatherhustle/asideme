# Variables

JavaScript is a programming language, which allows to write instructions for a computer to manipulate *memory*. That's why the foundation of *every* programming language is working with variables!

A **variable** is a named **pointer to a memory address**, at which a **value** is stored.

## Variable Diagram

```
variable → memory address → value
```

### Example 1: office variable
```
office → 8272719 → "123 Main St"
```

### Example 2: travelDistance variable
```
travelDistance → 127368 → 95
```
