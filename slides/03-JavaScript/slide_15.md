# Recap

1. Ecma International publishes **EcmaScript**.
2. An organization follows the specification to build a **JavaScript engine**.
3. The JavaScript engine is incorporated into a browser.

Some time later...

4. You type JavaScript code into a browser **console**.
5. The JavaScript engine translates your code into machine code.
6. The computer follows the instructions to update something in memory.
7. The new contents in memory are then shown to you via the browser console.
