# Data types

If the type changes, then the same binary will be interpreted as something wildly different!

## Diagram Structure

```
variable ──→ type ──→ value
                
lunch ──→ 1000 ──→ 11010101110101
           └────────────┘
                  ↓
              🛵 (scooter emoji)
```

### Components:
- **variable**: "lunch"
- **type**: 1000 (binary representation)
- **value**: 11010101110101 (binary representation)
- The value connects to a green scooter icon, illustrating how the same binary data is interpreted differently based on its type
