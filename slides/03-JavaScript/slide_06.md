# Programming language

A **markup** language allows us to store *content* in memory for a computer to read.

A **programming language** allows us to store *instructions* in memory for a computer to not only *read*, but also to *write* new things into memory.

A file that is written in a programming language is called a **program**. It is a sequence of instructions that a computer follows to manipulate memory.
