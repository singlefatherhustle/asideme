# Browser Console

You can interact with the browser's JavaScript engine by using the **browser console**. The console is an *interface* that lets you input JavaScript code, runs the code, and then returns the result. This type of interface is often referred to as a **REPL**.

- **Read** = Take input from the user (a line of code).
- **Eval** = Evaluate or execute the input code.
- **Print** = Output the result of the evaluation.
- **Loop** = Repeat the process for the next input.

## Browser Console Example

The screenshot shows a DevTools console from www.fullstackacademy.com/tech-bootcamp with the following visible code:

```javascript
5 + 3
8
```

This demonstrates the REPL in action:
- **Read**: `5 + 3` is entered
- **Eval**: The expression is evaluated
- **Print**: `8` is returned as the result
- **Loop**: Ready for the next input
