# Data types

Every variable also has a **type**, which is extra information stored in memory that tells the computer how to interpret the 1s and 0s of that variable's value.

## Diagram: Variable Structure

```
variable → type → value
```

## Example: lunch variable

```
lunch → 1101 → 11010101110101
            ↓
         [sandwich image]
```

The diagram shows:
- **variable**: lunch
- **type**: 1101 (binary representation of the type identifier)
- **value**: 11010101110101 (binary representation of the actual value)
- The type information tells the computer to interpret these 1s and 0s as a sandwich
