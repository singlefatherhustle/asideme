# Machine code

Computers actually don't understand programming languages! So how does a program work?

It must be **translated** into a language that computers do understand: **machine code**.

This translation process is called **compilation** or **interpretation**.¹

## Diagram: JavaScript to Machine Code

```
index.js (JavaScript)
    ↓
[compiled/interpreted]
    ↓
machine code:
1. Look at memory address 50
2. Write a 0
3. Move to memory address 84
4. Write a 1
```

---

¹: There's a slight difference related to who does the translation and when the translation happens!
