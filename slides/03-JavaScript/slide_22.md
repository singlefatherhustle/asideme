# Primitive data types

JavaScript comes with a few **primitive data types**, which are built into the language.

- **string** = a sequence of characters to represent text
  - "pie", "Hello 123", "😀"
- **number** = numeric data
  - 5, -3, 0.28
- **boolean** = true or false
- **null** = *intentional* absence of data
- **undefined** = *current* absence of data
