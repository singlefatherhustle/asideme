# Runtime environment

We will be using the **browser** to translate and run our JavaScript programs into machine code.

In other words, the browser is our **runtime environment**.

How do we provide a JavaScript file to the browser? The same way as CSS: via an HTML file!

## Diagram: File Structure and Browser Relationship

```
index.js ──┐
           │
           ├──→ index.html ──→ browser
           │
index.css ─┘
```

**Files shown:**
- index.js
- index.html
- index.css

**Flow:** The index.js and index.css files are referenced in index.html, which is then sent to the browser.
