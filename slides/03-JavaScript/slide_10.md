# ECMAScript

Ecma International (European Computer Manufacturers Association) is a nonprofit standards organization that publishes a **programming language specification** named **ECMAScript**.

You can think of this specification as a blueprint for a programming language. It dictates things such as the language's syntax and grammar rules, or special keywords and their intended behaviors.

Ecma International will periodically update the specification to include new features.
