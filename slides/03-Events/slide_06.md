# Events

An **event** is a signal that is **fired** by the browser whenever something significant happens (like a user interaction).

For example, when the user moves their mouse wheel, the browser fires a "scroll" event.

## Diagram: Event Flow

```
user
(stick figure)
     |
     |--move mouse wheel--> browser
                            (globe icon)
                               |
                               |
                            scroll
                            (label box)
```

The diagram shows:
- A user (stick figure) on the left
- An arrow labeled "move mouse wheel" pointing right to the browser
- The browser (represented by a globe icon)
- An arrow pointing up from the browser to a box labeled "scroll"
