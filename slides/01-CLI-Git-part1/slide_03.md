# Version Control

## Definition

A **version control system** allows us to track *multiple* different versions of a file by recording a history of events and changes to that file.

## Flowchart: Morning Routine Decision Tree

```
alarm rings, wake up
        ↓
   leave the house
     ↙          ↘
forget keys    remember the keys
    ↓                ↓
run back to house   get to work early
for keys
    ↓
late for work
```

### Flowchart Structure

- **Start node:** "alarm rings, wake up"
- **Decision node:** "leave the house" branches into two paths
  - **Left path:** "forget keys" → "run back to house for keys" → "late for work"
  - **Right path:** "remember the keys" → "get to work early"
