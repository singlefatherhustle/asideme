# Clone

If a repository already exists on GitHub, we **clone** it down in order to make a *local* repository that we can work in. This will connect our local repository with the remote repository.

## Command

```bash
git clone <url>
```

## Diagram

A cloud shape labeled "GitHub (Remote)" contains a yellow folder icon labeled "Repository". An arrow labeled "git clone" points from this remote repository to another yellow folder icon labeled "Repository" next to a laptop/computer icon, which together represent the "Local" environment.
