# Recap

1. `git checkout -b feature` to make a new feature branch.
2. Change some files in a repository.
3. `git add <file>` to add those changes to the staging area.
4. `git commit -m "message"` to commit those changes.
5. `git switch main` to switch back to the main branch.
6. `git merge feature` to merge in the changes from the feature branch.
7. Repeat from step 1!
