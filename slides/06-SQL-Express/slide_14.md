# The app receives a request

A client sends a POST request with headers and a body to /paintings.

## Diagram

```
client → POST /paintings → Express app
```

**Components:**
- **client**: Computer/device icon on the left
- **POST /paintings**: Hexagonal arrow shape in the middle representing the HTTP POST request
- **Express app**: Cloud shape on the right representing the server application

**Flow**: Client sends request → Express app receives request
