# Review

## What is the difference between a client and a server?

A client sends requests to a server. A server sends responses to a client.

## Client-Server Communication Diagram

```
client                          server
   |                               |
   |-------- request ------------->|
   |      Can I get XYZ?           |
   |                               |
   |<------ response --------------|
   |         OK! XYZ               |
   |                               |
```

**request**: Can I get XYZ?

**response**: OK! XYZ
