# Review

## What format should the resource paths of a RESTful API follow?

```
/resources/ID/subresources/ID/sub-subresources/ID/…
```

## Resource Path Examples

| Path | Resource |
|------|----------|
| /libraries | All the libraries |
| /libraries/3 | Library #3 |
| /libraries/3/books | All books in library #3 |
| /libraries/3/books/82 | Book #82 in library #3 |
| /libraries/3/books/82/chapters | All chapters in book #82 in library #3 |
