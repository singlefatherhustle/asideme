# Full-stack development

Working on **both** the frontend **and** the backend is referred to as **full-stack development**.

## Architecture Diagram

### client/frontend
- Web app

### server/backend
- API
- Database

**Connections:** The Web app communicates bidirectionally with the API, which in turn communicates bidirectionally with the Database.
