# Review

## What is a data type?

A data type is extra information stored in memory that tells the computer how to interpret the 1s and 0s.

## Diagram: Variable Structure with Data Type

```
variable → type | value
```

```
lunch → 1101 | 110101011101101
         ↓
      [sandwich image]
```

The diagram shows:
- A variable named "lunch"
- A type field containing "1101" (binary representation of the data type)
- A value field containing "110101011101101" (binary representation of the actual data)
- An arrow pointing down from the binary value to an image of a sandwich, indicating how the computer interprets the binary data
