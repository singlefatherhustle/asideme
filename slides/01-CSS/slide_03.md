# What is CSS?

**CSS** (Cascading Style Sheets) is a stylesheet language used to define the visual appearance and layout of HTML elements on a webpage.

## Visual Examples

### Example 1: HTML Only
- Heading: "Hello world!"
- Text: "This website is only displaying HTML."
- Image: Globe icon
- Text: "If you need to contact us please use this link."

### Example 2: HTML and CSS
- Heading: "HELLO WORLD" (styled with colors and effects)
- Background: Dark blue with large decorative earth/globe illustration
- Text: "This website is displaying HTML and CSS."
- Text: "If you need to contact us please use this link."
