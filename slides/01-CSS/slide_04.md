# Selectors

A **selector** determines which elements to style.

## Type Selector

A **type selector** (tag name selector, element selector), selects all elements of a certain type.

```
span, em, h1
```

## Class Selector

A **class selector** selects all elements with a certain class and is denoted by `.`

```
.tip, .info, .warning
```

## ID Selector

An **ID selector** selects a single element with a certain ID and is denoted by `#`

```
#mailbox, #front-door, #home
```
