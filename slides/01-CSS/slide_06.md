# CSS Syntax

Let's put it all together!
A CSS **rule** contains **selectors** followed by the **body** (denoted by `{}`).
The body contains **declarations**, which are **property:value** pairs.

A CSS file is a sequence of CSS rules.

## Code Example

```css
selector(s) {
    property: value; /* declaration */
    property: value; /* declaration */
}
```
