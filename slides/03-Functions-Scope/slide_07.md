# Arguments

How do we provide input to a function?
By **passing in an argument** when the function is called.
The same function can be called with *different* arguments to the same parameter.

```js
function greet(name){
    console.log(`Hi! My name is ${name}!`);
}
```

```js
greet("Xyzzy");  // Hi! My name is Xyzzy!
greet("Wobble"); // Hi! My name is Wobble!
greet("Thud");   // Hi! My name is Thud!
```
