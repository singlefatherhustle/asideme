# Parameters and arguments are different

A parameter is a **variable** that is defined when a function is **declared**.
An argument is a **value** that is provided to a parameter when a function is **called**.

## Diagram

```
"Xyzzy" ──────────────────┐
                           │

                   (robot) │
                           ▼
         ┌──────────────────────────────┐
         │                              │
         ▼                              ▼
      argument                      parameter
         │                              │
         ▼                              ▼
      value                          variable
         │                              │
      "Xyzzy"                         name
```

**Elements shown:**
- Function name: `greet`
- Argument: `"Xyzzy"` (the value passed when function is called)
- Parameter: `name` (the variable defined in function declaration)
- Relationship: The argument "Xyzzy" is a value that maps to the parameter variable `name`
