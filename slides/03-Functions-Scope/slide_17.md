# Variables get put on the stack

When we declare a variable or a function, it is added to the **stack**. Entries in the stack are organized like a *stack* of plates.

```javascript
const foo = 5;
const bar = "hello";
function wobble(){}
```

## Diagram: Stack vs Heap

**Stack:**
- wobble (top)
- bar
- foo (bottom)
