# Review

## Question
How is an *argument* different from a *parameter*?

## Answer
An argument is the *value* that is assigned to a parameter.

## Diagram
```
"Xyzzy" ──→ name ──→ value
   ↓          ↓         ↓
 greet    parameter   variable
```

### Components:
- **greet**: function name (represented by robot icon)
- **name**: parameter
- **"Xyzzy"**: argument (the string value being passed)
- **argument**: the value passed to a function
- **parameter**: the variable that receives the argument
- **variable**: where the value is stored
