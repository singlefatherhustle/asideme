# Command Syntax

You start typing the **command** after the **prompt**. A command's behavior can be modified by **flags** or **arguments**, which come after the command and are separated by spaces.

## General Structure

```
prompt | command | _ | flags | _ | arguments
```

## Example

```
demo@fsa ~ % | ls | _ | -al | _ | documents/
```
