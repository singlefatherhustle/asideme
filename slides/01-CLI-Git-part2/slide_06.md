# Directory Structure

All of the data on a computer is organized into files and folders. This organization is called a **directory structure**, and is often visually represented like this:

## Visual Diagram

```
                    [Folder]
                   /        \
              [Folder]      [Folder]
              /      \      /      \
          [File]  [Folder] [File]  [File]
                    /
                [Folder]
```

The diagram shows a hierarchical tree structure where:
- A top-level folder branches into two subfolders
- The left subfolder contains one file and one nested folder
- The nested folder contains another folder
- The right subfolder contains two files
