# Logical AND

The **AND** operator `&&` evaluates to `true` if and only if *both* operands are truthy. Otherwise it evaluates to `false`.

## Truth Table

| Expression | Result |
|---|---|
| `true && true` | `true` |
| `true && false` | `false` |
| `false && true` | `false` |
| `false && false` | `false` |
