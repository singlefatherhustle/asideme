# Truthy or Falsy?

When a value is converted into a boolean, it is either **truthy** or **falsy**.
In other words, it is **coerced** into either `true` or `false`.

## Truthy and Falsy Values Table

| Falsy | Truthy |
|-------|--------|
| `false` | `true` |
| `0` | everything else! |
| `""` | |
| `null` | |
| `undefined` | |
| `NaN` | |
