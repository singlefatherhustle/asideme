# Logical OR

The OR operator `||` evaluates to `true` if *at least one* operand is truthy. Otherwise it evaluates to `false`.

| Expression | Result |
|---|---|
| `true \|\| true` | `true` |
| `true \|\| false` | `true` |
| `false \|\| true` | `true` |
| `false \|\| false` | `false` |
