# Example

## Question
What is the final value of n?

## Code

```javascript
let n = 5;
n = n + 2;
n = 10 * n - 3;
```

## Solution

```javascript
n = 5;
n = n + 2;
    n = 5 + 2;
    n = 7;
n = 10 * n - 3;
    n = 10 * 7 - 3;
    n = 70 - 3;
    n = 67;
```

## Answer
**n = 67**
