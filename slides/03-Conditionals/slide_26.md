# Syntactic sugar

**Syntactic sugar** refers to *syntax* that allows us to write some code in a shorter and "sweeter" form.

For example, the following three lines of code all do the same thing: assign the *value* n+1 to the *variable* n.

```
n++;
// is syntactic sugar for
n += 1;
// is syntactic sugar for
n = n + 1;
```
