# If...else statements

Every `if...else` statement **must** start with `if`.
You can *optionally* chain as many `else if`s as you want.
`else` is also optional, but it **must** go at the end.

## Structure Diagram

| Component | Requirement |
|-----------|-------------|
| `if (condition) {}` | required |
| `else if (condition) {}`<br>`else if (condition) {}`<br>`else if (condition) {}` | as many as you want (0+) |
| `else {}` | optional at the end |
