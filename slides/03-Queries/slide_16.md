# How do we send a request?

Stay tuned! There's one more topic that we need to cover before we're ready to see the code, and that has to do with the fact that all internet communication is *slow*.

Running JavaScript code is really, really *fast*. Operations like adding two numbers or logging something to the console can be executed in less than a millisecond.

In comparison, a client with an average internet connection might need to wait more than 100 milliseconds to receive a response.
