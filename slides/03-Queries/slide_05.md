# Clients request responses from servers

All communication over HTTP follows the **request-response** format.
One computer, called the **client**, sends a **request** for some information.
The other computer, called the **server**, sends a **response** with that information.

## Diagram: Request-Response Flow

```
client                          server
  |                               |
  |-------- request -------->     |
  |      "Can I get XYZ?"         |
  |                               |
  |     <------- response --------|
  |         "OK! XYZ"             |
```

**Elements:**
- **client**: Computer on the left sending the request
- **server**: Computer on the right sending the response
- **request**: Arrow pointing right labeled "Can I get XYZ?"
- **response**: Arrow pointing left labeled "OK! XYZ"
