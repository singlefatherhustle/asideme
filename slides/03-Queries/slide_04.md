# Communication across the internet

The internet is a lot of computers sending 1s and 0s back and forth to each other. How do they know to interpret these 1s and 0s the same way? By agreeing on a **protocol**.

The protocol that we will be working with is **HTTP: Hyper Text Transfer Protocol**.

## Diagram: HTTP Communication Network

Four computers arranged in a square formation with bidirectional connections between them labeled "HTTP". The connections show:
- Top-left computer connected to top-right computer
- Top-left computer connected to bottom-left computer
- Top-right computer connected to bottom-right computer
- Bottom-left computer connected to bottom-right computer (labeled "HTTP")
- Additional diagonal connections crossing between top-left to bottom-right and top-right to bottom-left
