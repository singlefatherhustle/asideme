# Example

## Question
What other data structure have we seen before that also looks like this?

## Answer
A linked list!

All linked lists are trees, but not all trees are linked lists.

## Diagram
A tree structure with four nodes connected linearly:

```
    A (root)
    ↓
    B
    ↓
    C
    ↓
    D
```

Each node is represented as a circle containing its label (A, B, C, D). Arrows point downward indicating the parent-child relationships in sequence.
