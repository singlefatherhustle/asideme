## What is a linked list?

A linked list is a linear data structure where each node contains a value and a pointer to the next node in the list.

## Linked List Structure Diagram

```
head                                                    tail
  |                                                      |
  v                                                      v
┌────┬───┬─┐          ┌────┬───┬─┐          ┌────┬───┬─┐
│    │ A │ •  ------→ │ B  │ • ├─┐    ----→ │ C  │ • │ │
└────┴───┴─┘          └────┴───┴─┘          └────┴───┴─┘
```

The diagram shows:
- **head**: pointer to the first node (A)
- **tail**: pointer to the last node (C)
- Each node contains a value (A, B, C) and a pointer (•) to the next node
- Arrows indicate pointer relationships between nodes
