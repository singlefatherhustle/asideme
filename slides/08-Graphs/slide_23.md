# Edges

## Definition

A **directed graph** is a graph where edges have orientations. In other words, an edge points *from* one vertex to another.

An **undirected graph** is a graph where edges *don't* have orientations.

## Visual Examples

### Undirected Graph
```
Vertices: A, B, C
Edges: A-B, A-C, B-C
(edges shown as lines without arrows)
```

### Directed Graph
```
Vertices: D, E, F
Edges: D→E, D→F, E→F
(edges shown as arrows with direction indicators)
```

The diagram shows:
- **Left side (undirected graph)**: Three vertices (A, B, C) connected by plain lines forming a triangle
- **Right side (directed graph)**: Three vertices (D, E, F) with directed edges: D→E, D→F, E→F
