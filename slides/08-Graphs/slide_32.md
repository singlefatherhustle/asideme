# Example

## Definition

A **graph** is a collection of **vertices** (nodes) and **edges**.

## Question

Is this a graph?

**Yes! There are vertices [A,B,C,D] and edges.**

**All linked lists are trees, and all trees are graphs, so all linked lists are also graphs.**

## Diagram

```
A (node)
↓
B (node)
↓
C (node)
↓
D (node)
```

The diagram shows a directed graph structure with four vertices (A, B, C, D) connected by directed edges (arrows) flowing downward from A → B → C → D.
