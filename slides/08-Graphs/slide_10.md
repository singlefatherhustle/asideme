# Example

A **tree** is a collection of nodes that follow these rules:

- Each node can point to other nodes (**children**).
- Each node can only have one **parent**.
- A node cannot be its own ancestor.

## Is this a tree?

**Yes! It follows all 3 rules.**

## Diagram

```
        A
       /|\
      / | \
     B  C  D
```

**Structure:**
- Node A (root) has three children: B, C, and D
- Nodes B, C, and D each have exactly one parent (A)
- No circular relationships exist
