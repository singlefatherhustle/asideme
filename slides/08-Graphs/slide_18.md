# Example

A **tree** is a collection of nodes that follow these rules:

- Each node can point to other nodes (**children**).
- Each node can only have one **parent**.
- A node cannot be its own ancestor.

## Is this a tree?

**Yes! It follows all 3 rules.**

## Diagram

```
    A
    ↓
    B
    ↓
    C
    ↓
    D
```

The diagram shows a linear tree structure with:
- Node A at the root (top)
- Node B as a child of A
- Node C as a child of B
- Node D as a child of C
- Arrows indicating parent-child relationships flowing downward
