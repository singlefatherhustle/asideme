# Which one should you use?

If you just want to iterate through **all** vertices in a graph, then it doesn't matter. The time complexity of both algorithms is **O(V+E)**.

If you are **searching for** something, then

- **DFS** is better if you think the target is **far** from your starting vertex.
- **BFS** is better if you think the target is **close** to your starting vertex.
