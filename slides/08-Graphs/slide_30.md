# Example

A **graph** is a collection of **vertices** (nodes) and **edges**.

## Is this a graph?

**Yes! There are vertices [A,B,C,D] and edges.**

**All trees are graphs, but not all graphs are trees.**

## Diagram

```
       A
      /|\
     / | \
    B  C  D
```

The diagram shows:
- Node A at the top (root)
- Three directed edges (arrows) pointing downward from A
- Nodes B, C, and D at the bottom level
- Arrows directed from A → B, A → C, A → D
