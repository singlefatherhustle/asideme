# Adjacency Matrix

The space complexity of an adjacency matrix is O(V²), where V is the number of vertices.

## Time Complexities

What are the time complexities of:

- checking if an edge exists? **O(1)**
- getting the weight of an edge? **O(1)**
- adding an edge? **O(1)**
- removing an edge? **O(1)**
- adding a vertex? **O(V²)**
- removing a vertex? **O(V²)**
- getting all neighbors of a vertex? **O(V)**
