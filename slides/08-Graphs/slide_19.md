# Example

## Question

What other data structure have we seen before that also looks like this?

## Diagram

A linear linked structure with four nodes connected by directional arrows pointing downward:

- **Node A** (top) → arrows down
- **Node B** (second) → arrows down
- **Node C** (third) → arrows down
- **Node D** (bottom)

Each node is represented as a circle with a letter inside. Arrows indicate the direction of the link from one node to the next in sequence.
