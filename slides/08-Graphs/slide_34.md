# Adjacency matrix

An **adjacency matrix** is a 2D array where the value at index `ij` represents the weight of the edge from vertex `i` to vertex `j`.

## Graph Representation

A directed weighted graph with 4 vertices (0, 1, 2, 3):

- **Vertex 0** → Vertex 1 (weight: 53)
- **Vertex 0** → Vertex 3 (weight: 86)
- **Vertex 1** → Vertex 2 (weight: 12)
- **Vertex 3** → Vertex 1 (weight: 24)
- **Vertex 2** has no outgoing edges
