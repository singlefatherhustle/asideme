# Special trees

There are more specific kinds of trees that impose different rules.

For example, a **binary tree** is a tree where each node can have *max 2* children.

## Binary Trees vs Non-Binary Trees

### Binary Trees
Three examples of valid binary tree structures:

1. Single node (A only)
2. Node A with one child B
3. Node A with two children B and C

### Not Binary Tree
Example showing an invalid binary tree (highlighted in pink):
- Node A with three children: B, C, and D

This violates the binary tree rule since node A has more than 2 children.
