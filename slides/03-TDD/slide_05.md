# Enter: Node.js

**Node.js** is an alternative runtime environment that allows us to execute JavaScript code *without* going through HTML and the browser. It is powered by the V8 engine.

## Diagram

```
index.js → Node.js → V8 engine
                ↓
            (connected)
```

**Components shown:**
- **index.js** — JavaScript file (document icon)
- **Node.js** — Runtime environment (green hexagonal logo with "JS")
- **V8 engine** — JavaScript engine (blue circular logo with wings)

The diagram illustrates the flow: index.js file is processed by Node.js, which uses the V8 engine to execute the code.
