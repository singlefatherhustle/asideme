# Happy paths

A **happy path** is a test case where the code takes known input, executes without any errors, and produces an expected output.

We expect **multiply** to:
- be a function.
- take two numbers as input.
- return a single number.
- return 50 when given 5 and 10.
- return 12 when given 2 and 6.
