# Review: browser runtime environment

JavaScript files are attached to an HTML file. The browser parses the HTML and executes the JavaScript code with its JavaScript engine.

## Diagram: Browser Runtime Environment Flow

```
index.js → index.html → browser → JS engine
                           ↓
                    (contained within browser)
```

**Components:**
- **index.js** — JavaScript file
- **index.html** — HTML file
- **browser** — Runtime environment (globe icon)
- **JS engine** — JavaScript execution engine (orange puzzle piece icon)

**Flow:** The JavaScript file is included in the HTML file, which is loaded by the browser. The browser's JavaScript engine then executes the JavaScript code.
