# If not both, then which one?

In a 1-1 relationship, the foreign key can go in *either* table.

## Which one do you choose?

Think about the *has* and *belongs to* statements between the two tables.

- 1 **foo** *has* 1 **bar**
- 1 **bar** *belongs to* 1 **foo**

Then the foreign key should go in **bar**.
