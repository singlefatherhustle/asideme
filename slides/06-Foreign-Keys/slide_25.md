# Inner join

There are different types of join queries. The default join is an **inner join**, which only returns records that have matching values in **both** tables.

## table 1

| Column |
|--------|
| A      |
| B      |
| C      |
| D      |

## table 2

| Column |
|--------|
| A1     |
| A2     |
| C3     |
| D1     |
| E1     |

## (INNER) JOIN Result

| Column | Column |
|--------|--------|
| A      | A1     |
| A      | A2     |
| C      | C3     |
| D      | D1     |
