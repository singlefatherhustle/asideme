# Array indexes

An array's **elements** are stored in a very specific order.
An element's **index** indicates where it is in the array.

```javascript
const numbers = [3, 2, 1, 1, 0];
```

## Array visualization

| Index | 0 | 1 | 2 | 3 | 4 |
|-------|---|---|---|---|---|
| Element | 3 | 2 | 1 | 1 | 0 |
