# Why use an array?

Arrays allow us to store related data in a **single** variable, rather than creating new variables for each piece of data.

## Code Comparison

| Without Array | With Array |
|---|---|
| `const vegetableA = "carrots";`<br>`const vegetableB = "cabbage";`<br>`const vegetableC = "lettuce";` | `const vegetables =`<br>`["carrots", "cabbage", "lettuce"];` |

---

**All content must be stored in a single variable.**
