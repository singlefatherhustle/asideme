# How do we use React?

React code is written in **JSX**, which is a special *dialect* of JavaScript. It has to be **transpiled** into **vanilla JS** before a JS engine is able to understand and run the code.

## Process Diagram

### Without Transpilation (Incorrect)
JSX → ??? → JS engine ❌

### With Transpilation (Correct)
JSX → **transpile** → JS → JS engine ✓ OK!
