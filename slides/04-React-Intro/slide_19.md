# Recap

## Key Points

1. Write **React** components in **JSX** files.
   a. React automatically re-renders components when state changes.

2. Use **Vite** to **transpile** and **bundle** these files into a vanilla JS file.

3. The browser executes that JS file to build a document.

## Workflow Diagram

**Dev** (stick figure)
- writes → JSX files (6 React atom icons arranged in 2 rows of 3)
- uses → **Vite** (purple and yellow icon)
  - **transpiles & bundles** → **JS** (vanilla JavaScript file)

The diagram shows:
- Developer writes multiple JSX files
- Vite processes these files (transpilation and bundling)
- Output is a single vanilla JavaScript file
- This JS file is executed by the browser to build the document
