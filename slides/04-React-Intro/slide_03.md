# Review

1. JS code is executed by the browser's JS engine to manipulate the DOM.
2. The browser notices changes in the DOM and relays that to the render engine.
3. The render engine updates the document according to the changes in the DOM.

## Diagram: JavaScript Execution Flow

```
index.js → JS engine → DOM → render engine → document
```

- **index.js** (document file icon)
- **JS engine** (orange puzzle piece icon)
