# Grid track

A **track** refers to the space *between* two lines, and can be either horizontal (**row**) or vertical (**column**).

## Grid Diagram

```
     1    2    3    4    5
1 |____|____|____|____|____|
  |         row              |
2 |____|____|____|____|____|
  |    | col|    |    |
3 |____|umn |____|____|____|
  |    |    |    |    |
4 |____|____|____|____|____|
```

The diagram illustrates:
- **Row**: A horizontal track spanning from column 1 to column 5 between rows 1 and 2
- **Column**: A vertical track spanning from row 2 to row 4 between columns 2 and 3
