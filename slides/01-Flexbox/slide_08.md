# Media queries

**Media queries** are another tool for responsive design.

**@media rules** are rules that apply to other *rules*, rather than HTML elements. The targeted rules will only apply if the viewport matches the media query.

## Code Example

```css
@media (width > 500px) {
  body {
    background-color: aquamarine;
  }
  h1 {
    font-style: italic;
  }
}
```
