# Media query breakpoint

A **media query breakpoint** refers to a specific viewport width where a media query will take effect and change how a site is styled.

There is no standard set of breakpoints, but a common baseline you can use is:

- **300px** for mobile devices
- **600px** for tablets
- **900px** for laptops
- **1200px** for desktops
