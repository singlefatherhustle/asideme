# Grid cell

A **grid cell** is the smallest unit you can have in a grid. It is the space between four intersecting grid lines.

## Diagram

```
     1    2    3    4    5
   +----+----+----+----+
 1 |    |    |    |    |
   +----+----+----+----+
 2 |    | cell   |    |
   +----+----+----+----+
 3 |    |    |    |    |
   +----+----+----+----+
 4 |    |    |    |    |
   +----+----+----+----+
```

The diagram shows a grid with 5 columns and 4 rows. A single grid cell (highlighted in blue) is located at the intersection of columns 2-3 and rows 2-3, demonstrating that a grid cell is bounded by four intersecting grid lines.
