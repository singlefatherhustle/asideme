# Grid

Grid allows you to change track sizes, make more tracks, or span a grid item over multiple tracks, and move grid items anywhere within the grid!

## Grid Layout Diagram

```
     1    2    3    4    5
   +----+----+----+----+----+
 1 |         A         |    |
   +----+----+----+----+----+
 2 |    |         B         |
   +----+----+----+----+----+
 3 | C  |    |    |    |    |
   +----+----+----+----+----+
 4 |    |    |    |    |    |
   +----+----+----+----+----+
```

### Grid Items:
- **A** (light blue): Spans columns 1-4, row 1
- **A** (light yellow): Spans column 5, row 1
- **B** (light blue): Spans columns 3-5, rows 2-3
- **C** (light blue): Spans column 1, rows 3-4
- Various light yellow cells filling remaining spaces
