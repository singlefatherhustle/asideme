# Grid

Just like flex containers, a **grid container** automatically rearranges its children according to its **grid lines**, which are numbered and can be vertical or horizontal.

## Grid Diagram

```
Line
     1    2         3    4         5

1    |____|_________|____|_________|
     |    |         |    |         |
2    |____|_________|____|_________|
     |    |         |    |         |
3    |____|_________|____|_________|
     |    |         |    |         |
4    |____|_________|____|_________|
```

The diagram shows a grid container with:
- **Vertical lines** (columns): numbered 1, 2, 3, 4, 5
- **Horizontal lines** (rows): numbered 1, 2, 3, 4
- **Grid cells**: formed by the intersection of vertical and horizontal lines
- **Shaded area**: represents the grid container spanning from line 1 to line 5 horizontally and line 1 to line 4 vertically
