# Arrays

- use square brackets to separate values with commas
- [`carrots`, `cucumbers`, `potatoes`, `tomatoes`, `yarrow`]
- [1, 3, 5, 7, 2, 12, 4]
- [`hello`, 54, true, 23, `goodbye`]
- use indexes starting at 0 to decide where items are in the array

## Array Manipulation - use ARRAY METHODS to add/edit/delete from the array

- Put a dot after the array name followed by the method name. Then invoke the method.
- push -> adds to the END of an array and returns the new length
- pop -> removes the LAST item in an array and returns that element
- unshift -> adds to the BEGINNING of an array and return the new length
- shift -> removes the FIRST item in an array and returns that element
- slice -> makes a copy of a portion of the array
- splice -> add/edit/delete from an array

## Array Properties, Access, and Setting by Index

- length -> how many items are in the array
  - LENGTH IS ALWAYS EQUAL TO THE LAST INDEX PLUS ONE!!!
- accessing an index -> arrayName with square brackets and index number between the brackets
  - arrayName[indexNumber]
- setting by index number -> same as accessing but now assigning
  - arrayName[indexNumber] = value

## Convert Strings to Arrays

- stringVariable.split() -> turns a string into an array based on a passed in separator
