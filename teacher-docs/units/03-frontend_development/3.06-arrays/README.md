## Learning Objectives

- Use indexes to access and modify individual elements of an array
- Use indexes to access individual characters of a string
- Iterate through the indexes of an array with a `while` or `for` loop
- Iterate through the elements of an array with a `for..of` loop
- Iterate through a nested array with a nested loop
