# Loops -> allow us to repeat the same code multiple times

- for loops -> go through a block of code a set number of times
  - for(initialization; condition; afterthought) { what you want to do for each item in the array }
  - initialization -> executed one time BEFORE the code starts looping
  - condition -> defines a condition for continuing to exectue the code block
  - afterthought -> executed AFTER each iteration of the loop
  - intialization -> condition -> code block -> afterthought -> condition -> code block -> afterthought...
- while loops -> go through a block of code while a certain condition is true
  - while(condition) { code here }
- BE CAREFUL! This an lead to an infinite loop if the condition never fails
