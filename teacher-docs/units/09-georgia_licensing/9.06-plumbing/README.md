# Plumbing — Georgia Study Guide

**Unit:** Georgia Licensing
**Topic:** Plumbing (Journeyman, Master Class I, Master Class II)

---

## License Classes

Georgia plumbing separates **Journeyman**, **Master Class I**, and **Master Class II**:

| License | Scope |
|---|---|
| Journeyman | Statewide journeyman plumbing |
| Master Class I | **Single-family dwellings**, **one-level dwellings designed for not more than two families**, and **commercial structures not exceeding 10,000 sq ft** |
| Master Class II | **Unrestricted** |

## Qualification Requirements

| License | Experience | Notes |
|---|---|---|
| Journeyman (statewide) | At least **1 year** plumbing experience | Pass with at least **70** |
| Master Class I | **5 years** plumbing experience with at least **2 years as a licensed journeyman plumber** | |
| Master Class II | Same as Class I **plus commercial or industrial plumbing experience** | |

## Reference List (2025) — Critical Distinction

The official 2025 plumbing reference list is especially important because it distinguishes the exam format:

- **Journeyman Plumber exam is open-book.**
- For all other plumbing exams, the listed references **may not** be used during the examination.

References allowed for the master exams:
- 2018 Georgia Gas Code
- 2018 Georgia Plumbing Code
- ADA / employer materials
- Blueprint reading
- Plumbing math
- Plumbing technology
- Georgia 811 excavator material
- Tax materials
- OSHA Part 1926

## What Plumbing Questions Hinge On

In practice and on the exam, plumbing questions often hinge on:
- **Scope** (which license class)
- **Who may supervise** (master, business ownership rules)
- **Vent/drain logic** (where the code lives)

## Safe Venting Workflow

1. Identify the fixture and required trap.
2. Confirm **trap arm sizing and distance limits** under the applicable plumbing code.
3. Determine whether the code allows **individual**, **common**, **wet**, **circuit**, **stack**, or **combination** venting for the setup.
4. Route the vent so it rises properly, avoids prohibited offsets below flood level where applicable, and ties back into the vent system correctly.
5. Check **cleanouts**, **slope**, and **fixture-unit load**.
6. Final-check against gas-code and local inspector expectations where the installation also includes gas appliances.

## Sample Exam-Style Questions

**Q:** What is the scope of a Georgia Class I Master Plumber license?
**A:** It is limited to plumbing involving **single-family dwellings**, **one-level dwellings designed for not more than two families**, and **commercial structures not exceeding 10,000 square feet**.

**Q:** Which Georgia plumbing license exam is open-book according to the board's 2025 reference list?
**A:** The **Journeyman Plumber** exam.

## Real-Life Scenario

**Scenario:** A licensed journeyman wants to open a plumbing company and contract directly with the public.
**Answer:** Georgia's plumbing FAQ says the board does **not register plumbing companies**, but **only licensed Master Plumbers may have a business — not Journeymen**.
