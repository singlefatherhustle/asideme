# Plumbing Exam Structure

## At a glance

| Class | Open-book? | Time | Pass |
|---|---|---|---|
| Journeyman | Yes (per 2025 ref list) | ~180 min | 70 |
| Master Class I | No | ~240 min | 70 |
| Master Class II | No | ~270 min | 70 |

The key distinction: the **Journeyman exam is open-book**, but **all master exams are closed-book**.

## Approved references (2025)

For Journeyman (open-book) and as study material for masters:

- 2018 Georgia Plumbing Code (IPC + GA amendments)
- 2018 Georgia Fuel Gas Code
- ADA / employer materials
- Blueprint reading
- Plumbing math
- Plumbing technology
- Georgia 811 excavator material
- Tax materials
- OSHA Part 1926
- NASCLA Business and Law

## Topic distribution

Approximate:

```
Plumbing code (GA): 25%
Drainage systems:    15%
Water supply:        15%
Venting:             12%
Gas piping:          10%
Backflow prevention:  6%
Plumbing math:        6%
Business/Law:        10%
Safety/OSHA:          1%
```

## Master Class II distinction

Class II includes additional topics:

- Medical gas systems
- Industrial process plumbing
- Pure water systems
- Specialty backflow assemblies
- Hospital-grade plumbing

## What gets tested heavily

```
Common test areas:
1. Drainage sizing (DFU calculations)
2. Vent sizing and configuration
3. Trap arm distances
4. Pipe material selection by application
5. Pipe sizing for water supply
6. Cross-connection prevention
7. Backflow device types and applications
8. Sample math problems
```

## Reference book preparation (Journeyman)

For the open-book Journeyman exam:

- Tab the IPC chapters: 3 (general), 5-9 (drainage, vents, water supply, gas)
- Mark key tables: fixture units, drainage pipe sizing, vent sizing
- Print clean tabs (no handwritten notes)
- Practice navigating quickly under time pressure

## Calculator allowed

Scientific calculator (no text-entry, no graphing). Required for:

- DFU calculations
- Pipe sizing
- Backflow protection
- Hydraulics math

## Common exam day errors

- Confusing IPC vs UPC (Georgia uses IPC)
- Forgetting to look up tables (master candidates without book lose points)
- Misreading scale on fixture-unit tables
- Confusing supply and drainage pipe sizing (different tables, different rules)
- Spending too long on math while leaving easier code questions unanswered
