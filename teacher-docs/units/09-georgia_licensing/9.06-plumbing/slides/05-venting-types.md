# Venting Types

## Why venting matters

Vents serve three critical functions:

1. **Equalize pressure** in the drainage system (prevents siphoning of traps)
2. **Allow air to enter** behind moving water (smooth flow)
3. **Exhaust sewer gases** out of the building

A drain without proper venting will siphon, gurgle, and stink.

## Five venting strategies

### 1. Individual Vent

Each fixture has its own vent:

```
   ┌───── Vent
   │
   │
   ├── Trap arm
Trap
   │
Fixture drain
```

Simplest, safest, most reliable. Larger homes use individual vents.

### 2. Common Vent

Two fixtures share one vent:

```
Fixture A ─── Trap ─── Trap arm ──┐
                                  ├── Common vent
Fixture B ─── Trap ─── Trap arm ──┘
```

Each fixture must be on opposite sides of the shared vent fitting. Common for lavatory + tub sharing a vent.

### 3. Wet Vent

A drain pipe ALSO serves as a vent for another fixture downstream:

```
Lavatory   Toilet
   │         │
   │         │
   └── Wet vent ── (vented)
         │
       Drain
```

Code limits which fixtures can be wet-vented (low DFU only). Common in residential bath groups.

### 4. Circuit Vent (Branch Vent)

A single vent serves multiple fixtures on a horizontal branch:

```
                  Circuit vent
                       │
                       │
Fix 1 ──┐    Fix 2 ─── Fix 3 ─── Fix 4 ── Branch drain
        │                                    │
        └────────────────────────────────────┘
```

Used in commercial settings with multiple fixtures in a row.

### 5. Stack Vent / Air Admittance Valve (AAV)

Stack vent: continuation of vertical drainage pipe up through roof.

AAV (one-way valve):
- Allows air IN to relieve negative pressure
- Closes against positive pressure (no sewer gas)
- Listed devices only
- Cannot be hidden behind wall (must be accessible)
- Limited applications per IPC (e.g., island sinks)

## Vent sizing

Vents sized based on DFU served:

```
Vent < 100 feet long:
   1.25 inch:  6 DFU
   1.5 inch:  10 DFU
   2 inch:    24 DFU
   2.5 inch:  48 DFU
   3 inch:    102 DFU
```

For longer vents, size up.

## Vent termination

Through-roof venting requirements:

```
- Minimum 6 inches above roof
- 10 feet from any door/window/intake within 3 feet horizontal
- Through roof, capped with weatherproof cover
- 3 inches minimum diameter where it penetrates roof (in cold climates to prevent ice closure)
```

## Branch interval

In multi-story buildings:

```
Each story is one "branch interval"
Vertical drainage system requires:
   - Stack vent extending through roof
   - Possibly relief vents at each branch
   - Yoke vent between stack and trap arm at lower fixtures
```

## Common venting errors

- Trap arm too long for pipe size (siphoning)
- Wet venting fixtures that aren't code-allowed
- Single vent serving too many DFU
- Vent terminated through wall too close to window/door
- Inadequate roof clearance (less than 6 inches above roof)
- Using AAV where it's not allowed (most fixtures still require true vent)
- Missing relief vent in multi-story stack
