# Backflow Prevention

## What backflow is

Reverse flow of water from a contaminated source back into the potable water supply. Two ways it happens:

- **Back-siphonage**: negative pressure in supply line draws contamination back
- **Back-pressure**: contaminated water at higher pressure pushes back into supply

Both contaminate the public water system, sickening many people. Backflow prevention is required by code.

## Categories of hazard

| Category | Description | Required protection |
|---|---|---|
| **High hazard** | Contaminants that can kill (sewage, chemicals, pesticides) | RPZ (most stringent) |
| **Low hazard** | Aesthetic contamination (taste, smell) | Various lower-grade devices |
| **No hazard** | None | None |

## Backflow prevention devices

### Air Gap

Simplest method. Physical separation between potable outlet and contaminated surface:

```
   Faucet
     │
     │ Air gap (≥ 2× outlet diameter, min 1 inch)
     ▼
   Sink rim
   Water surface
```

Used at: sinks, dishwashers (high-loop), water heaters.

### Atmospheric Vacuum Breaker (AVB)

For low-hazard, non-pressure applications. Cannot be under continuous pressure (12 hours max).

```
Water flow → AVB → Outlet
   When pressure drops, air enters AVB to break vacuum
```

Used at: hose bibs, lawn sprinklers (with restrictions), industrial sinks.

### Pressure Vacuum Breaker (PVB)

For low-hazard, under continuous pressure:

```
Water flow → PVB → Outlet
   PVB allows air entry on negative pressure
   Allowed under continuous pressure (unlike AVB)
```

Used at: lawn irrigation, swimming pool fillers.

### Double Check Valve Assembly (DCVA)

Two check valves in series; for low-hazard applications:

```
Water → CV1 → Test cock → CV2 → Outlet
```

Used at: fire sprinkler systems, low-hazard commercial connections.

### Reduced Pressure Zone (RPZ)

Highest-rated; for high-hazard applications:

```
Water → CV1 → Relief valve → CV2 → Outlet
        Relief opens if pressure differential changes,
        dumping water to drain rather than letting backflow occur
```

Used at: chemical injection systems, hospital fluid lines, fire system with antifreeze, irrigation with fertilizer injection.

## Test and inspection

Backflow preventers must be tested:

```
Initial: at installation
Annual: by certified backflow tester
Repair: after any modification or failure
```

Test results filed with the water utility. Non-compliant devices result in supply disconnection.

## Common backflow installation requirements

### Hose bibs

```
Hose bib with vacuum breaker (HB-VB)
OR
Hose bib with separate vacuum breaker installed
```

Required on every hose bib in new construction.

### Lawn irrigation

```
PVB or RPZ at irrigation supply (depending on hazard)
Located outside, accessible for testing
Must be specifically rated for irrigation
```

### Fire sprinkler

```
RPZ at supply (high-hazard if antifreeze used)
DCVA at supply (low-hazard, dry system without antifreeze)
```

### Commercial dishwashers

```
Air gap upstream (not a backflow device but functions as one)
Or specific commercial dishwasher protection
```

## Common backflow errors

- Hose bib without vacuum breaker
- Lawn irrigation without backflow device
- Missing annual tests on installed devices
- Wrong device type for the hazard level
- Backflow device installed below grade (sometimes prohibited)
- Cross-connection on potable supply (e.g., direct connection of bidet without proper protection)
