# Georgia Lien Law

## What a mechanic's lien is

A legal claim against real property for unpaid work or materials. Allows contractors, subcontractors, and material suppliers to enforce payment by encumbering the property.

```
Property owner doesn't pay → contractor files lien → property cannot be 
sold or refinanced until lien is paid (or invalidated)
```

## Who can file

In Georgia:

```
- General contractor (in direct contract with owner)
- Subcontractor (in contract with GC)
- Sub-subcontractor (in contract with sub)
- Material supplier (delivered to project)
- Equipment rental supplier (if equipment was used on the job)
```

## Critical deadlines

```
1. Notice of Commencement: filed by OWNER, public record
   Establishes the start date for lien rights

2. Notice to Contractor / Notice of Furnishings:
   Required if you don't have direct contract with owner
   Must be filed within 30 days of first labor or materials supplied
   Sent to owner AND to GC of record

3. Mechanic's Lien Filing:
   Must be filed within 90 days of last work date
   In the county records where the property is located

4. Foreclosure Lawsuit:
   Must commence within 1 year of the lien filing date
```

Missing ANY deadline forfeits the lien. These are strict.

## The 90-day rule explained

```
Last day of substantial work on the project: April 1
Lien filing deadline: July 1 (90 days later)
   Even if you continue minor punch-list work after April 1,
   the 90-day clock typically starts from substantial completion
```

What counts as "work":

- Labor performed
- Materials installed
- NOT typically: warranty visits, minor repairs after closeout

## What the lien contains

```
- Description of property (legal description preferred)
- Name of owner
- Total amount owed
- Brief description of work performed or materials provided
- Date(s) of work
- Signature + sworn under oath
```

A properly recorded lien:

- Becomes a public record
- Shows up on title searches
- Prevents clear title until resolved

## How an owner responds

Owner options when a lien is filed:

```
1. Pay the amount → contractor releases the lien
2. Dispute the amount → may sue or counterclaim
3. File bond to discharge → property is released; bond stands in
4. Ignore → after 1 year without lawsuit, lien expires
```

## Common lien mistakes

### Filing too early
```
Lien filed before substantial work completion
Result: lien may be premature, invalidated
```

### Filing too late
```
Lien filed beyond 90 days
Result: lien is void
```

### Wrong property description
```
Vague description "John Smith's house"
Result: lien fails for description; cannot be enforced
```

### No notice of furnishings (sub)
```
Subcontractor didn't file notice within 30 days
Result: lien rights forfeited
```

### Inflated amount
```
Lien filed for more than actually owed
Result: not enforceable for the excess; may invite counter-suit
```

## Anti-lien provisions

Some property is exempt from mechanic's liens:

- Public buildings (state, federal, municipal)
- Owner-occupied homestead in some circumstances
- Properties with specific contract language excluding liens

For public projects: use a Miller Act bond (federal) or state public works bond instead.

## The lien waiver

Owners often require lien waivers as payments are made:

```
Conditional waiver: voids when payment is received in full
Unconditional waiver: voids immediately upon signing

Always sign conditional waivers, not unconditional, unless 
you have payment in hand.
```

## Common lien law errors

- Missing the 90-day filing deadline
- Filing without 30-day notice (subs/suppliers)
- Wrong property description
- Filing lien on exempt property
- Signing unconditional waiver before payment cleared
- Failing to file foreclosure suit within 1 year
- Inflating the lien amount
