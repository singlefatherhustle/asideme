# Project Management Workflow

## The 9-step contractor workflow

```
1. Confirm license class and scope
2. Estimate and bid
3. Contract execution
4. Permit / code planning
5. Schedule subs and order materials
6. Coordinate specialty trades
7. Site safety / OSHA compliance
8. Inspections and punch work
9. Closeout, lien-waiver, final billing, records
```

Each phase has specific documentation, deliverables, and risks.

## Phase 1 — Scope confirmation

Before bidding:

- Match your license class to the work
- Identify required specialty subcontractors (electrical, plumbing, HVAC)
- Determine permit requirements
- Verify your insurance covers the scope

## Phase 2 — Estimating and bidding

Standard practice:

```
Cost = Direct labor + Direct materials + Subcontractor costs + Equipment
Overhead allocation: 10-20% on top of cost
Profit margin: 10-25% on top of cost + overhead
Total bid: Cost + Overhead + Profit
```

Markup vs margin (a common confusion):

```
Cost: $50,000
Selling price: $65,000
Profit: $15,000

Markup (% of cost): $15,000 / $50,000 = 30%
Margin (% of selling price): $15,000 / $65,000 = 23%
```

## Phase 3 — Contract execution

Standard contract elements:

- Names and addresses of all parties
- Scope of work
- Total contract amount and payment schedule
- Schedule (start date, substantial completion, final completion)
- Warranty terms
- Change order procedure
- Lien rights / waiver language
- Insurance requirements
- Dispute resolution
- Signatures + dates

## Phase 4 — Permit and code planning

For permitted projects:

```
1. Submit drawings for plan review
2. Pay permit fees
3. Address plan review comments
4. Permit issued
5. Begin construction (cannot start before permit)
```

Each phase typically has its own inspection:

- Footing/foundation
- Framing (pre-drywall)
- Rough plumbing/electrical/HVAC
- Insulation
- Drywall
- Final

## Phase 5 — Scheduling and procurement

Critical path method (CPM):

```
- Identify all tasks
- Determine task duration
- Sequence tasks based on dependencies
- Identify critical path (tasks that determine project end date)
- Float (slack) on non-critical tasks
```

Material lead times:

```
Standard items (lumber, drywall, basic doors): 1-2 weeks
Custom items (windows, specialty doors, cabinets): 6-12 weeks
Long-lead (HVAC equipment, switchgear, elevators): 12-24+ weeks
```

Order long-lead items first.

## Phase 6 — Trade coordination

Subcontractor sequencing:

```
1. Site work (excavation, utilities)
2. Foundation
3. Framing
4. Rough HVAC, plumbing, electrical
5. Insulation
6. Drywall
7. Interior finishes (paint, trim, doors)
8. Trim HVAC, plumbing, electrical
9. Flooring
10. Final finishes
```

Trade conflicts cost money — coordinate to prevent interference.

## Phase 7 — Safety and OSHA

Per 1926 standards:

- Daily safety meeting
- PPE compliance check
- Fall protection above 6 feet
- Excavation safety (trenching)
- Lift safety
- LOTO procedures

Document compliance — if an inspector asks, you should be able to show it.

## Phase 8 — Inspections and punch list

Punch list (final walkthrough items):

- Minor cosmetic flaws
- Trim adjustments
- Color touch-ups
- Operating instructions to owner

These shouldn't include structural or major mechanical items — those should have been addressed already.

## Phase 9 — Closeout

```
Final paperwork:
- Certificate of occupancy / final inspection sign-off
- Lien waivers from all subs and suppliers
- Warranty documents
- Manufacturer's manuals
- As-built drawings
- Final billing
- Customer signoff on substantial completion
```

The closeout documentation is the contractor's protection in disputes.

## Common project management errors

- Skipping the formal contract (verbal agreements)
- Starting without permit
- Inadequate change order procedure (verbal additions)
- No critical path schedule (just "estimates")
- Trade coordination by phone vs documented schedule
- Skipping safety meetings
- No final closeout documentation
- Verbal lien waiver instead of written
