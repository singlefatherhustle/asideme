# Print Reading Reference

## What "print reading" means

Print reading = the ability to interpret electrical construction drawings. On the exam, you'll be given diagrams showing:

- Single-line diagrams
- Schematic diagrams
- Wiring diagrams
- Floor plans with electrical layout
- Riser diagrams
- Conduit and cable routing plans

Questions ask you to identify components, trace circuits, or count items shown.

## Standard symbol set

Memorize these recurring symbols. They appear on almost every print-reading question.

```
●         Outlet (general)
⊕         Duplex receptacle
⊕GFI      GFCI receptacle
⊕WP       Weatherproof receptacle
$         Single-pole switch
$3        Three-way switch
$4        Four-way switch
○         Ceiling outlet (incandescent)
○⊙        Ceiling outlet (fluorescent)
▣         Disconnect switch
▥         Motor controller
M         Motor
T         Transformer
G         Generator
———       Branch circuit (concealed in ceiling)
- - -     Branch circuit (concealed in floor)
- ⋯ -     Exposed wiring
```

## Single-line diagram conventions

Single-line (one-line) diagrams show power flow:

```
   Utility
     │
   [Meter]
     │
   [Main breaker]
     │
   [Main panel]
   /  |  \
[Br1][Br2][Br3]   ← branch circuits
```

Each segment is labeled with conductor size, conduit, and circuit ID.

## What's typically tested

- Reading a panel schedule and identifying available capacity
- Tracing a circuit from panel to load
- Counting outlets / lights on a single circuit
- Identifying voltage and phase from the diagram
- Spotting a code violation in the layout (e.g., GFCI missing in wet location)

## Common errors

- Confusing single-line (power) with schematic (control) diagrams
- Missing the difference between solid (concealed) and dashed (exposed) line types
- Counting a 3-way switch pair as two separate switches in fixture-count questions
