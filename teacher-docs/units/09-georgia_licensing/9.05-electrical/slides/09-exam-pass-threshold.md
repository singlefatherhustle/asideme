# Pass Threshold and Retake Rules

## The score you need

A passing score on the Georgia electrical contractor exam is **70**. The exam is scored on a 0–100 scaled basis.

```
              passing line
    0 ─────────────────────────── 70 ───── 100
                                  ▲
                            70 = minimum
```

## What happens if you fail

| Attempt | Wait time before retake |
|---|---|
| 1st fail | Re-apply through PSI immediately; pay exam fee again |
| 2nd fail | Same — re-apply through PSI |
| **3rd attempt or later** | **One-year wait** required after the last failed attempt |

The one-year mandatory wait kicks in if you've failed twice. The board uses this period to encourage candidates to address gaps in preparation before burning more attempts (and more fees).

## What "one year" means

It's a hard 365-day wait from the date of the second failed attempt. The clock does not reset for partial-year prep. The board does not grant exceptions.

Use the year:

- Take a NEC-focused continuing education course
- Get more field experience in areas you struggled on
- Practice with code-book navigation drills
- Take a paid commercial exam-prep course (Mike Holt, IEC, IBEW)

## How the scoring actually works

The exam is computer-based at PSI. After you finish:

- Score is calculated immediately
- Pass/fail is shown on the screen
- A printed score report is given before you leave the testing center
- If you failed, the report shows percentage correct in each content area (so you know where to focus)

## Score release timing

- Pass: a candidate ID number is generated, license application is forwarded to the board for final approval
- Fail: only the score report is generated, no application moves forward

## Strategy implications

- **Don't underprepare and burn a free attempt.** The application fee + exam fee + your time is non-trivial.
- **If you fail once,** review the score report carefully and address the weakest content area before re-sitting.
- **Don't fail twice in a row.** The 1-year wait is painful and delays starting your business.
