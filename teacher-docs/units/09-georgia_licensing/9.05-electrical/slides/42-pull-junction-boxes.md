# Pull and Junction Boxes (NEC 314.28)

## When pull/junction boxes are required

Pull boxes must be installed:

- When the **conduit fill** is exceeded
- When the number of **bends** between pull points exceeds 360°
- For practical pulling distances on long runs

## Sizing — straight pulls

For conductors **#4 AWG or larger** in straight pulls through a box:

```
Box length ≥ 8 × diameter of the largest raceway
```

Example:
```
Straight pull with 2" raceway:
  Box length ≥ 8 × 2 = 16 inches
```

## Sizing — angle and U pulls

For angle pulls or U-pulls:

```
Distance from raceway entry to opposite wall ≥ 6 × largest raceway diameter
PLUS the sum of all other raceway diameters on the same wall
```

Example: 2" angle pull with two 1.5" raceways on the same wall:
```
Box dimension ≥ (6 × 2) + 1.5 + 1.5 = 15 inches
```

## Distance between raceway entries on the same wall

```
Raceway entries enclosing the same conductors:
   Distance ≥ 6 × largest raceway diameter
```

This prevents conductors from making a sharp U-turn inside the box that would damage the insulation.

## Box accessibility — 314.29

Pull and junction boxes must remain accessible. **Hidden behind a wall, ceiling, or surface that requires demolition to reach is not accessible.**

Acceptable:
- Above a removable ceiling tile
- In an unfinished basement
- In a service mechanical room
- Behind a removable cover plate

Not acceptable:
- Behind a permanent finished wall
- Above a rigid ceiling without an access panel

## Box covers — 314.41

- Must be securely fastened
- Listed for the application
- Provide the same enclosure rating as the box itself (e.g., weatherproof for outdoor)

## Worked example: deciding when to add a pull box

You're running 1-1/2" EMT for 80 ft with three 90° bends so far. Time to plan ahead:

```
Bends so far: 3 × 90° = 270°
Limit: 360° between pull points
Remaining bend budget: 90°

If the run still needs more bends, install a pull box now.
If straight from here, you can finish without one.
```

## Common pull-box errors

- Installing in inaccessible location (behind drywall)
- Undersized for the raceway diameter
- Multiple raceways entering on the same wall without observing the spacing rule
- Pulling 4/0 wire through a 4×4 junction box (too small for the conductor diameter)
- Forgetting that the SAME conductors entering and exiting count for the "same wall" calculation
