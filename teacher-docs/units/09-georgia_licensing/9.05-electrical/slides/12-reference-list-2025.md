# Approved Reference List (2025)

## What you can bring

The official 2025 reference list for the Georgia electrical contractor exam allows these — and only these — in the testing room:

| Reference | Notes |
|---|---|
| **National Electrical Code (NEC)** | Current Georgia-adopted edition (2020 NEC through 2025; 2023 NEC effective Jan 1 2026) |
| **American Electricians' Handbook** | Croft / Summers, current edition |
| **Ugly's Electrical References** | Pocket reference |
| **Printreading text** (one) | Approved title — see bulletin |
| **NASCLA Contractors Guide to Business, Law and Project Management** | For the Business & Law portion |
| **IRS Employer's Tax Guide (Publication 15)** | Current year |
| **OSHA 29 CFR Part 1926** | Construction industry standards |
| **Georgia State Electrical Code** | If published separately; otherwise NEC + GA amendments |

## What you CANNOT bring

- Loose-leaf notes or photocopies
- Handwritten content of any kind in your code books
- Sticky notes or Post-its (use printed tabs)
- Calculator with text-entry capability
- Phone, smartwatch, electronic device
- Snacks, food, gum
- Hats, sunglasses (unless prescription)

## Tabbing strategy

A useful tab layout for the NEC:

```
Chapter 1   Definitions, working spaces, conductor IDs
Chapter 2   Branch circuits, feeders, services, OCP, grounding
Chapter 3   Wiring methods (NM, AC, EMT, raceway types)
Chapter 4   Equipment, switches, receptacles, motors
Chapter 5   Special occupancies (505, 517, etc.)
Chapter 6   Special equipment (transformers, capacitors)
Chapter 7   Special conditions (emergency, optional systems)
Chapter 9   Tables (conduit fill, conductor properties)
Annex C/D   Conduit fill tables by conductor type
```

## What changes 1 Jan 2026

Georgia adopts the **2023 NEC** statewide effective **January 1, 2026**. Key shifts:

- Updated GFCI/AFCI requirements
- Revised energy storage system rules (Article 706)
- New EV charging install provisions
- Updated tamper-resistant receptacle scope

If you sit the exam in 2025, expect the 2020 NEC. From January 2026 onward, expect 2023.

## Source check

Always confirm the live reference list on the PSI candidate information bulletin for the **specific exam date** you're scheduled. The board updates the list periodically.
