# Class I — Scope Details

## The 200 A / single-phase rule

The Class I scope limit is literally: *"single-phase electrical installations not to exceed 200 amperes at the service drop or service lateral."* Two constraints, both must hold.

```
                    Utility transformer
                          │
                    Service drop / lateral
                          │
                  ┌───────┴───────┐
                  │  Meter base    │  ←─ measurement point for the 200 A cap
                  └───────┬───────┘
                          │
                   Main panel (200 A max for Class I)
                          │
                ┌─────────┼─────────┐
              Branch    Branch    Branch
```

## What Class I CAN do

- Single-family dwellings ≤ 200 A service
- Duplexes where each unit has its own ≤ 200 A service
- Small light-commercial buildings on a single-phase service
- Service upgrades from 100 A → 150 A → 200 A
- New branch circuits, sub-panels, dedicated circuits — all downstream of the limited service

## What Class I CANNOT do

- 400 A residential services (now common in larger homes)
- Three-phase service of any size
- Motor control centers with three-phase motors
- Industrial floor work
- High-rise residential where the building service is three-phase

## The decision tree

```
Does the installation use three-phase service?
    YES → Class II required
    NO  → Continue
Is the service > 200 A at the drop/lateral?
    YES → Class II required
    NO  → Class I sufficient
```

## Why this matters at bid time

A Class I contractor who unknowingly bids on a job with a 400 A residential service has a licensing problem before the first wire is pulled. Verify the service size in the spec or on-site **before** signing.
