# Georgia's Construction Industry Licensing Board Structure

## The umbrella

The Georgia Secretary of State's Office houses multiple professional licensing boards. For construction-related trades:

| Board | Trades covered |
|---|---|
| **Construction Industry Licensing Board** | Residential and General Contractors |
| **Electrical Contractors Division** | Electrical contractors |
| **Plumbing Examiners Board** | Plumbers (journeyman + master) |
| **Conditioned Air Contractors Division** | HVAC contractors |
| **Real Estate Commission (GREC)** | Real estate salespersons + brokers |

Each operates with its own rules and exam administration but coordinates through the Secretary of State's office.

## Common functions across boards

```
1. License issuance (initial)
2. Renewal administration (typically every 2 years)
3. Continuing education approval
4. Examination administration (typically through PSI)
5. Disciplinary actions
6. Complaint investigation
7. Statute interpretation
8. Rule-making (within statutory authority)
```

## Statutes and rules

For each trade:

```
Statute (legislative law):
- Georgia Code, Title 43 (Professions and Businesses)
- Chapter 14: Real Estate
- Chapter 40: Electrical Contractors
- Chapter 41: Plumbing
- Chapter 41A: Conditioned Air
- Chapter 41B: Construction Industry

Rules (board-issued regulations):
- Found in the Georgia Administrative Procedures
- More detailed than statute
- Subject to administrative rulemaking process
```

## Hierarchy of authority

```
1. Georgia Code (statute) — passed by legislature
2. Board rules — adopted by the licensing board
3. Board policies — interpretive guidance
4. Specific licensing actions — applied to individual licensees
```

A board cannot make rules contradicting statute. Statutes change only by legislation.

## Online presence

Each board:

- Maintains a website with rules, forms, fees
- Online renewal portal
- Approved-provider lookup
- Disciplinary action public records
- Verification of license status (for the public to confirm a contractor is licensed)

## Public licensee lookup

Any member of the public can search:

```
- License status (active, expired, revoked)
- License number
- Disciplinary history
- License class / type
- Business name (for incorporated practitioners)
```

This is at the consumer level — homeowners verify their contractor before hiring.

## Complaint process

```
1. Consumer files complaint with appropriate board
2. Board investigates (records request, possible interview)
3. Board determines if violation occurred
4. If yes: disciplinary action (warning, fine, suspension, revocation)
5. Licensee has right to appeal
6. Final order entered in licensee's record
```

## Coordination between boards

When a violation crosses trade lines:

- Each board handles violations in its trade
- Shared violations (e.g., fraud) may go to multiple boards
- Criminal referrals to district attorney for serious cases
- Civil cases proceed separately

## Common trade framework errors

- Looking at wrong board's rules for a question
- Assuming one trade's licensing rules apply to another
- Missing inter-trade coordination requirements
- Operating without active license in any required trade
- Working on a project requiring multiple licenses without all properly licensed
