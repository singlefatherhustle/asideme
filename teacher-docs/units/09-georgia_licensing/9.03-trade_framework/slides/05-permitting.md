# Permitting Across Trades

## When a permit is required

Generally required for:

| Type | Permit needed? |
|---|---|
| New construction (any structure) | Yes |
| Additions to existing building | Yes |
| Major remodel (load-bearing changes) | Yes |
| Electrical service changes | Yes |
| New HVAC installation | Yes |
| Plumbing additions or major repairs | Yes |
| Roof replacement (most jurisdictions) | Yes |
| Pool installation | Yes |
| Detached structures > certain size | Yes |
| Window replacement (most jurisdictions) | Yes |
| Cosmetic interior work (paint, flooring) | Usually no |
| Minor repairs in kind | Usually no |

## Who can pull permits

```
For trade work:
- Licensed contractor in that trade
- Sometimes the property owner (with specific restrictions)
   - Owner usually limited to single-family + own occupancy

NOT allowed:
- Sub-contractor working under another contractor (they're "on" the GC's permit)
- Unlicensed help, even with supervision
- Self-described "handyman" without trade license
```

## Permit application

```
Standard requirements:
1. Property address + parcel ID
2. Owner name + contact
3. Contractor license number
4. Scope description (typically with drawings)
5. Estimated cost / value of work
6. Application fee
```

## Inspection sequence

For most permit work:

```
1. Footing / foundation (if applicable)
2. Rough-in (framing, plumbing, electrical, HVAC rough)
3. Insulation (if applicable)
4. Drywall (before paint)
5. Final inspection
```

Each phase has its own inspection. Cannot proceed to next phase until previous is approved.

## Permit fees

Typical Georgia residential permit fees:

```
Permit fee:           $20-100 (base, varies by jurisdiction)
Plan review fee:      $50-500 (depending on complexity)
Per-trade additions:  $25-200 (electrical, plumbing, HVAC additional)

Plus: contractor's bond + insurance verification fees
```

For commercial work, fees scale with project value.

## Penalties for unpermitted work

```
Penalty: typically 2× the permit fee that would have been charged
Plus: required to obtain permit retroactively
Plus: opening of work for inspection (drywall removed, etc.)
Plus: structural review may be required
Plus: occupancy delayed if certificate of occupancy is needed
```

In serious cases, the work may need to be removed and redone.

## Contractor vs owner-builder

Some jurisdictions allow homeowners to pull permits for their own work:

```
Owner-builder must:
- Own and occupy the property
- Do the work themselves (not subcontract to others)
- Sign waiver acknowledging they're not a licensed contractor
- Not sell the property for specified period after completion

Owner cannot:
- Hire unlicensed help on their own permit
- Take advantage of owner exemption for rental or speculative properties
- Pull permits for someone else's property
```

## Inspector authority

Inspectors:

- Have authority over their jurisdiction
- Can stop work for safety violations
- Can require additional documentation
- Can require corrections before final inspection
- Have discretion on minor deviations

Contractors should:

- Maintain professional relationships
- Address callouts promptly
- Document everything
- Not argue with inspectors on-site (escalate properly)

## Multi-trade coordination

When a project spans multiple trades:

```
General contractor pulls primary permit
   ↓
Subcontractors operate under that permit OR pull their own trade permits
   (varies by jurisdiction)

Best practice:
- GC tracks all subs' permits
- Each sub's licensing verified
- Sub-trade inspections coordinated with GC schedule
- Final inspections after all trades complete
```

## Common permit errors

- Starting work without permit
- Underestimating value to avoid higher fees (caught at appraisal)
- Wrong contractor name on permit
- Skipping rough-in inspection
- Operating under expired permit (most permits expire 6-12 months)
- Closing walls without inspection of rough work
- Mid-project changes without amended permit
