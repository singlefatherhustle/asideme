# Disciplinary Actions

## How discipline arises

```
1. Consumer complaint to GREC or trade board
2. Audit finding (trust account, contract review)
3. Inspector report of violations
4. Other licensees reporting misconduct
5. Court records (lawsuits, judgments)
6. Routine renewal review
```

## Investigation process

```
1. Complaint or trigger event
2. Investigator assigned
3. Records request from licensee
4. Possible on-site visit
5. Interview with licensee
6. Investigator's report to board
7. Board determines if violation occurred
8. If yes: discipline + appeal opportunity
9. Final order entered in licensee's record
```

## Levels of discipline

| Level | What it is |
|---|---|
| **Warning** | Formal note in file; no other consequences |
| **Fine** | Monetary penalty (typically $500-$10,000) |
| **Probation** | License remains active but with restrictions |
| **Suspension** | License inactive for specified period |
| **Revocation** | License permanently terminated |

## What triggers what level

### Warning level

- Minor procedural violation (e.g., late renewal fee)
- First offense without harm
- Quickly corrected by licensee
- Cooperative response to investigation

### Fine level

- Substantive but minor violation
- Inadequate disclosure
- Procedural failures that didn't harm anyone
- Insurance lapse briefly
- Documentation deficiencies

### Probation level

- Significant violation but cure possible
- Repeated minor violations
- Trust account issues without conversion
- Unauthorized practice that's been corrected
- Required additional supervision or CE

### Suspension level

- Major violation
- Unauthorized practice in a different state
- Repeated discipline despite warnings
- Conversion of trust funds (if intent was unclear)
- Significant safety violations

### Revocation level

- Conviction of certain felonies
- Repeated revocations
- Pattern of fraud or dishonesty
- Substantial conversion of trust funds
- License obtained by fraud
- Repeated practice while suspended

## Notable Georgia disciplines

### Trust account violations

```
First-time minor commingling, corrected: warning + CE
Significant commingling: probation + restricted license
Conversion: revocation + criminal referral
```

### Unlicensed work

```
First offense: warning + cease and desist
Repeated: civil penalty up to $5,000 per violation
Significant ongoing operation: criminal referral
```

### Lien violations

```
Wrongful lien filing: civil penalty + possible damages
Pattern: probation + practice restrictions
```

### Code violations

```
Found by inspector + reported to board:
- Minor: educational notice
- Significant: fine + correction required
- Severe (safety): suspension + retraining
```

## Appeal process

Every licensee has the right to:

```
1. Receive notice of proposed discipline
2. Request a hearing
3. Present evidence and witnesses
4. Be represented by counsel (at own expense)
5. Cross-examine board's witnesses
6. Appeal final order to court
```

The hearing is typically before an administrative law judge.

## Public record

All final disciplinary actions:

- Become public record
- Searchable on GREC / trade board website
- Visible to potential customers
- Disclosable on license applications in other states
- Permanent (unless expunged through specific process)

## Recovery / reinstatement

After suspension:

```
1. Complete suspension period
2. Pay any outstanding fines
3. Complete any required CE
4. Submit reinstatement application
5. Board reviews and may impose probation
6. License restored (sometimes with restrictions)
```

After revocation:

```
- May be eligible to re-apply after specified waiting period
- Subject to board discretion
- Requires substantial demonstration of rehabilitation
```

## Common discipline avoidance

```
- Maintain insurance continuously
- Renew on time
- Complete CE before deadlines
- Maintain trust accounts properly (separate, reconciled monthly)
- Respond promptly to any board inquiry
- Disclose proactively if there's an issue
- Comply with all board orders
```

## What NOT to do

```
- Ignore complaints from customers (escalates to formal complaint)
- Fail to respond to investigator requests
- Practice while suspended
- Move clients to a personal trust account during board scrutiny
- Backdate documents to comply
- Threaten or harass investigators
```

These behaviors transform a minor issue into a major one.
