# Reciprocity Across Trades

## What reciprocity means

State-to-state agreement that licensees from one state qualify for accelerated licensing in another. Each state's reciprocity rules are unique.

## Reciprocity by trade

### Real Estate
```
Reciprocal states (subject to GREC updates):
   Florida, Alabama, Tennessee, South Carolina, 
   North Carolina (limited), West Virginia, Mississippi
   
Path: Apply with home-state license verification + Georgia-specific exam
```

### Electrical
```
Limited reciprocity:
   Some neighboring states have specific arrangements
   Most require Georgia license application + exam
   Master Electrician status from another state usually accepted
```

### Plumbing
```
Limited reciprocity:
   Tennessee, Alabama (specific arrangements)
   Master Plumber status from another state usually requires verification
```

### HVAC / Conditioned Air
```
Some reciprocity available with neighboring states
Class I master usually accepted; Class II may require exam
```

### General Contractor
```
NASCLA-accredited exam accepted nationally:
   Pass NASCLA once, qualify for general contractor in many states
   Each state still requires its own license application
   Each state's Business & Law exam still required
```

## How NASCLA changes the game

```
Single NASCLA exam = qualifying credential for:
   Georgia
   Alabama
   South Carolina
   Tennessee
   Louisiana
   Mississippi
   Virginia
   North Carolina
   Texas (with additions)
```

This is unique to general contractor licensing. Trade-specific licenses (electrical, plumbing, HVAC) don't have a parallel national exam.

## What reciprocity does NOT cover

- **Insurance**: must meet each state's specific minimums
- **Background check**: each state runs its own
- **Continuing education**: each state's CE counts only in that state
- **Specific renewals**: each state has its own renewal cycle
- **Disciplinary history**: triggers review in any new state

## The "two-license" arrangement

Common in metro areas spanning state lines:

```
Atlanta contractor often holds:
- Georgia electrical license
- Florida electrical license (cross-border work)

Each requires:
- Separate CE
- Separate insurance certificates
- Separate renewal cycles
```

This is common but operationally complex.

## Non-resident contractor

A contractor licensed in another state working in Georgia:

```
Cannot operate as general contractor or specialty contractor in GA 
   without Georgia license
   
Cannot pull permits in GA without proper GA licensing
   
Exception: brief consulting visits or referrals don't require licensure
   (similar to attorney "pro hac vice")
```

## Reciprocity request process

```
1. Apply to Georgia board with home-state license verification
2. Demonstrate current active license in home state
3. Pass Georgia-specific exam portion (varies by trade)
4. Pay fees
5. Background check
6. Bond or insurance per Georgia requirements
7. Find sponsoring Georgia broker / employer
```

## Reciprocity FAQ

- **Q: Does my home-state CE count?** No — only Georgia-approved CE.
- **Q: Can I operate in Georgia on my home-state license?** No — Georgia license required for active practice.
- **Q: My home-state license is restricted (e.g., due to discipline)** Reciprocity may be denied.
- **Q: I moved my business; do I need to re-apply?** Yes — new GA application required.

## Common reciprocity errors

- Operating in Georgia on out-of-state license alone
- Assuming all states with reciprocity have identical rules
- Skipping Georgia-specific exam portion
- Letting home-state license lapse, breaking reciprocity
- Mixing CE between states (only counts in the state where it was approved)
