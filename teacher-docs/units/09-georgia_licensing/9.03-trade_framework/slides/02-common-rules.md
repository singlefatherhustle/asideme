# Common Rules Across Trades

## Universal contractor requirements

All Georgia trade contractors share certain obligations:

| Requirement | Where applied |
|---|---|
| **Licensure required for compensated work** | Doing the trade for pay requires license |
| **Affirmative duty to maintain license** | License lapse = unlicensed work |
| **Insurance requirements** | GL minimum per trade + workers' comp if employees |
| **Permit pulled for permit-required work** | Most major work requires permits |
| **Disclosure of license number on advertising** | All advertising must show valid license number |
| **Final inspection of permitted work** | Required before customer occupancy |
| **Customer protection from misrepresentation** | Contractor cannot misrepresent qualifications |

## License renewal cycle

```
All trades: typically 2-year renewal cycle
Each trade has its own deadline (varies)
Renewal requires:
   - CE completion
   - Renewal fee payment
   - Continuing eligibility (no disqualifying changes)
```

## Continuing education

Required for all trade licenses:

```
Real Estate: 9 hours per 2-year cycle (specific mandatory topic each cycle)
Electrical: 8 hours per 2-year cycle (including NEC update content)
Plumbing: 8 hours per 2-year cycle
HVAC: 8 hours per 2-year cycle
General Contractor: 8 hours per 2-year cycle (Business & Law updates)
```

Approved-provider system; only approved hours count.

## Insurance baseline

Standard minimums across trades:

```
General Liability:
   Residential Basic: $300,000
   Most other classes: $500,000

Workers' Comp:
   Required when 3+ employees
   Excluded sole proprietors don't need it for themselves

Property:
   Optional but recommended
```

## Background check requirements

All trade licenses require:

```
Initial: LiveScan fingerprinting + state/FBI background check
Renewal: typically not required again
Disqualifying offenses (varies by trade):
   - Felony involving fraud, theft, dishonesty
   - Sex offense involving minors
   - License revocation in another state still pending
   - Some misdemeanors (case-by-case)
```

## Operating standards

Common to all trades:

```
1. Compliance with applicable code
2. Truthful representation to customers
3. Proper handling of funds (trust accounts where applicable)
4. Insurance maintained continuously
5. Sub-contractor compliance verification
6. Adequate supervision of unlicensed help
```

## Failure to maintain license

Consequences of license lapse:

```
- All work performed while lapsed = unlicensed work
- Customers may refuse payment or sue
- Insurance coverage may be voided
- Criminal liability for continued operation
- Difficulty reinstating after long lapse
```

## License vs registration

| Term | Meaning |
|---|---|
| **License** | Authorization to practice (with associated qualifications) |
| **Registration** | Information record-keeping; less stringent |

Most trades require licensing (with qualifications). Some specialty trades may only require registration.

Georgia generally requires LICENSING for:
- Electrical contractors
- Plumbing contractors (journeyman + master)
- HVAC contractors
- Residential / General contractors
- Real estate sales / brokerage

Lower-tier registration may apply to:
- Sub-trades
- Specialty consultants
- Inspectors (specific endorsement)

## Common framework errors

- Letting license lapse during work
- Sub-contracting to unlicensed help on permitted work
- Operating across multiple trades without proper license for each
- Missing CE hours at renewal
- Incomplete background check at initial application
- Failing to disclose previous license lapse on renewal
