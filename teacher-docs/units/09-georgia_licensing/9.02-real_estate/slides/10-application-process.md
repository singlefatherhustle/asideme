# Application Process

## The full application workflow

```
1. Complete 75-hour pre-license course
2. Receive course completion certificate
3. Apply to GREC (or apply through PSI exam first, depending on path)
4. Schedule and pass the licensing exam
5. Submit license application with exam results
6. Background check / fingerprinting
7. Find sponsoring broker
8. Receive license certificate
```

## Application packet

The application packet includes:

| Item | Notes |
|---|---|
| Application form | GREC current version |
| Pre-license course completion certificate | Within 1 year of issuance |
| Exam score report | From PSI |
| Background check / fingerprinting | LiveScan vendor |
| Sponsoring broker confirmation | Letter or form from broker |
| Application fee | Currently ~$170-200 |
| Photo ID copy | Government-issued |

## Background check

Required for all license applicants:

- LiveScan fingerprinting at approved vendor
- Cost: ~$60-80
- Processing: 1-2 weeks
- Disqualifying offenses: certain felonies (varies by case, GREC reviews)
- Some misdemeanors (especially fraud, dishonesty) flagged for review

If you have a criminal record, contact GREC **before** paying the application fee. They review case-by-case.

## Sponsoring broker

Before activating, you must have a sponsoring broker:

- Broker must be in good standing with GREC
- Sponsorship letter signed by qualifying broker
- Broker's brokerage information on file
- Commission split / agreement details (between broker and you, not GREC)

You cannot activate your license until a sponsoring broker confirms.

## License activation

Once GREC processes everything:

- License is "issued" but **inactive**
- Cannot practice until activated through broker
- Activation = your broker confirms affiliation
- License then becomes **active**

## Initial license term

```
Salesperson initial license: 4 years
Then: 2-year renewal cycles
```

The first 4-year period is longer because new licensees take post-license courses within the first year.

## Common application rejections

- Background check flagged for criminal history → review delays
- Pre-license course expired (over 1 year old) → must retake
- Sponsoring broker not confirmed → license issued inactive
- Application fees not paid → returned
- Missing pages or signatures → returned for completion

## Timeline expectations

- Pre-license course: 4-12 weeks (depending on pace)
- Background check: 1-2 weeks
- Application processing at GREC: 2-4 weeks
- Total from "start studying" to "active license": 8-16 weeks

## After issuance

```
First 12 months:
   Complete 25 hours of post-license courses
   
Every 2 years thereafter:
   9 hours of CE
   Renewal fee
```
