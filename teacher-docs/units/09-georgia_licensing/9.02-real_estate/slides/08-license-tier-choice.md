# Choosing Your License Tier

## The decision framework

```
                Start
                  │
         Do I have prior real
         estate experience?
         │              │
        No             Yes
         │              │
   Salesperson      Have 3+ years?
                       │
                      Yes
                       │
            Broker exam ready?
                       │
                 ┌─────┴─────┐
               Yes          No
                │           │
        Open own firm?      │
         │     │            │
        Yes    No           │
         │     │       Take broker
       Broker  Assoc   course + exam
       (own)   Broker
```

## When Salesperson is right

- First license in any state
- Want to work within an established brokerage
- Don't have 3+ years of licensed experience
- Don't want firm-level responsibilities
- Want lower upfront cost and faster path to closing first deal

## When Associate Broker is right

- Have 3+ years of licensed practice
- Want better commission split
- Want broker credential without operating a firm
- Still want firm support / supervision / shared resources
- Open to firm-level responsibilities later

## When Broker (independent) is right

- Want to operate your own brokerage
- Want maximum commission retention
- Comfortable with trust account responsibility
- Have business management experience
- Have capital for office, advertising, errors-and-omissions insurance

## Switching tiers

You can move up the ladder:

```
Salesperson → (3 yrs + course + exam) → Associate Broker → (open firm) → Independent Broker
```

Going DOWN is rare and requires:

- Surrendering broker license
- Re-affiliating as associate broker or salesperson (different rules per situation)
- Notice to GREC

## Cost comparison

```
Salesperson:
  Pre-license: $200-500 (course)
  Exam: $98
  Initial license: $125
  Yearly cost (CE + renewal): $100-200
  Total Y1: ~$525

Associate Broker (added cost vs Salesperson):
  Broker course: $400-800
  Broker exam: $98
  Higher CE requirements
  Total addition: ~$500-900

Independent Broker (added cost vs Associate Broker):
  Firm registration: $100-500
  Office lease + setup: $500-2000+
  Errors-and-omissions insurance: $300-1500/yr
  Trust account setup: $0-100
  Marketing budget
```

## License-tier vs commission-tier

Many large brokerages have **internal tier systems** independent of license type:

```
ABC Realty:
  New Agent (any license): 50/50 split
  Producer ($500k+ annual): 60/40 split
  Top Producer ($2M+ annual): 70/30 split
```

These internal tiers don't change the legal license — but they affect take-home pay significantly.

## What matters most

For someone entering the trade:

1. Get licensed as Salesperson — fastest start
2. Affiliate with a brokerage that provides training + reputation
3. Build a client base for 3+ years
4. Decide based on goals whether to upgrade

The Salesperson license is sufficient for the vast majority of agents who plan to work within a firm.

## Common tier-choice mistakes

- Going straight to Broker without operational experience (firm fails)
- Staying Salesperson too long without exploring Associate Broker upgrade
- Operating as Independent Broker without enough capital
- Switching firms too often, hurting reputation
