# Types of Agency

## Three primary relationships

| Agency type | Whose interests the broker represents |
|---|---|
| **Single Agency** | One party only — the broker represents either seller OR buyer |
| **Dual Agency** | Both parties — the broker (with consent) represents both seller AND buyer |
| **Designated Agency** | Each party — different agents at the same firm represent different sides |

## Single agency

The traditional and clearest agency relationship:

```
                  Broker
                /        \
            Seller        Buyer
        (represented)   (customer or
                         represented by 
                         a different firm)
```

**Seller agency**: broker represents the seller's interests exclusively. Buyer is the "customer."

**Buyer agency**: broker represents the buyer's interests exclusively. Seller is the "customer."

The broker owes fiduciary duties to their represented party only.

## Dual agency

When the same broker represents BOTH parties:

```
                  Broker
                 (Dual Agent)
                /          \
            Seller          Buyer
        (represented)   (represented)
```

**Hard rules**:

- Both parties must give **informed written consent**
- The broker cannot disclose confidential info from one to the other
- The broker cannot negotiate price — they can only facilitate communication
- Material facts about the property must be disclosed to both
- Negotiation is between buyer and seller directly

Dual agency is legal in Georgia but heavily regulated. Disclosure must be signed BEFORE the broker begins representing both sides.

## Designated agency

A modern variant:

```
                ABC Realty (firm)
                /            \
        Agent A              Agent B
        represents           represents
         Seller               Buyer
```

Same brokerage firm, but different agents at the firm represent each side. The qualifying broker oversees both but is not personally the agent for either.

This is often preferred over dual agency because:
- Each party gets a dedicated advocate
- Less risk of one party feeling their interests aren't represented
- Cleaner fiduciary duty for each agent

Georgia recognizes designated agency under specific firm policies. Both parties must consent.

## Customer vs client

```
Customer: a non-represented party in a transaction (other side of the deal)
Client:   the party the broker represents (owes fiduciary duties to)
```

Common scenario:

```
ABC Realty represents the seller.
A buyer walks in and asks ABC Realty to show them a house.
ABC Realty must:
   - Disclose they represent the seller
   - Tell the buyer: "We can show you the house, but we represent the seller."
   - The buyer is a CUSTOMER, not a client.
```

The buyer can:
- See the house
- Make an offer
- Negotiate (with broker assistance, but not advocacy)

But they cannot expect the agent to advocate against the seller (who is the client).

## Sub-agency

When a co-operating broker (different firm) shows a listed property:

- The listing broker still represents the seller
- The co-op broker may be a SUB-agent of the listing broker
- Sub-agent owes fiduciary duties to the seller, even though they're working with the buyer

Modern practice has largely eliminated sub-agency in favor of designated buyer agency.

## Common agency errors

- **Acting as dual agent without disclosure**: license suspension
- **Treating customer as client** (or vice versa): violates fiduciary duty
- **Confusing sub-agency**: most modern brokers reject sub-agency to avoid these violations
- **Not disclosing agency relationship**: required at first substantive contact
