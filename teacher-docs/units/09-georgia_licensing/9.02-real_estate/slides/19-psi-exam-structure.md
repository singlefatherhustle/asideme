# PSI Exam Structure

## Where you take it

PSI Services LLC administers the Georgia real estate licensing exam. Testing centers in:

- Atlanta
- Augusta
- Columbus
- Savannah
- Albany
- Macon

Appointments are scheduled through PSI's online portal after your pre-license course completion is verified.

## Salesperson exam at a glance

| Attribute | Detail |
|---|---|
| Vendor | PSI |
| Format | Computer-based, multiple choice |
| Total questions | 152 |
| Time limit | 4 hours |
| Passing score | 75% (state) and 70% (national portion) |
| Open-book | No |
| Calculator | Approved scientific calculator allowed |

## Exam structure

The Georgia real estate exam has TWO portions:

```
National portion: 80 questions, 70% to pass
   Topics: federal law, agency basics, contracts, math,
            valuation, finance, fair housing

State portion: 72 questions, 75% to pass
   Topics: Georgia license law, GREC rules, state-specific
            disclosures, agency under Georgia law,
            trust accounts, Georgia contracts
```

You must pass BOTH portions to be licensed. Failing one means re-taking that portion only.

## Score reporting

After completion:

- Pass/fail is shown on screen immediately
- Score report (showing percentage correct per topic) printed before leaving
- Pass: candidate ID issued; application proceeds
- Fail: only score report; you can schedule a retake

## Broker exam

For broker license:

- Similar structure to salesperson exam
- More advanced questions on broker operations, trust accounts, agency
- Longer (typically 200+ questions, 5+ hours)
- Passing standard similar (~70-75%)

## Common pass rates

- Salesperson first attempt: 65-72%
- Broker first attempt: 60-68%

Higher than many other professional licenses. The exam is passable with dedicated preparation, but failure rate is significant — don't go in unprepared.

## Time management

```
152 questions / 240 minutes = 1.6 minutes per question
```

Practice tests should match this pace. Use the first 90 minutes for the questions you know; reserve the last 60 minutes for questions you flagged for review.

## Common exam-day errors

- Arriving late (PSI is strict — no admittance after start time)
- Missing required ID (two forms; one photo)
- Not bringing approved calculator
- Cramming morning-of (better to be rested than over-studied)
- Not flagging difficult questions to revisit
- Spending too long on one question (move on, return later)
