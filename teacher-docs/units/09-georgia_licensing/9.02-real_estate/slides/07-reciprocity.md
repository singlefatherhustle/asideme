# Reciprocity — Specific Mechanics

## What reciprocity means

State-to-state agreement that licensees from one state may obtain a license in another state with reduced requirements. Each state's reciprocity is unique — Georgia's terms aren't automatic.

## Georgia's reciprocity policy

Georgia maintains reciprocity with a list of states whose licensing requirements are deemed substantially similar. As of recent years:

```
Florida, Alabama, Tennessee, South Carolina, North Carolina, 
West Virginia, Mississippi (subject to current GREC updates)
```

The list updates periodically. Always check GREC's current reciprocity table before relying on it.

## What reciprocity gets you

If your home-state is on the reciprocity list, your Georgia license application is **expedited**:

- Skip pre-license education (75 hours)
- Take only the Georgia-specific portion of the exam (about 30-40 questions vs the full 152)
- Pay Georgia application fee
- Provide license certification from home state
- Find Georgia broker affiliation

## What reciprocity doesn't get you

- Full waiver of exam (you still take the GA portion)
- Waiver of fees (full fee still required)
- Waiver of background check
- Independence (you still need a Georgia broker)

## The Georgia-specific exam portion

The 30-40 question portion covers:

- Georgia license law
- Georgia agency rules
- Georgia trust account regulations
- Georgia disclosure requirements
- Georgia-specific contract forms
- Recent GREC rules and amendments

Pre-license study materials specific to Georgia (Bert Rodgers, Career WebSchool, etc.) prepare candidates for this portion.

## Salesperson reciprocity

A salesperson from another state needs:

1. Active license in home state for at least 3 years (some states' rules vary)
2. Application + fee
3. Pass Georgia-specific exam portion
4. Sponsor by Georgia broker
5. Compliance with GA continuing education thereafter

## Broker reciprocity

Similar, but:

1. Must hold broker license in home state
2. Pass Georgia-specific broker exam portion
3. May practice as Georgia broker, but must have GA brokerage if operating independently
4. CE requirements specific to broker track

## When reciprocity won't work

- License in non-reciprocal state → full pre-license course required
- License lapsed in home state → reciprocity unavailable, full application
- Disciplinary action pending → reciprocity may be denied or delayed
- Criminal history → background check may require additional review

## The "two-license" arrangement

Some agents hold licenses in both Georgia and a neighboring state (e.g., Atlanta-based agent licensed in GA and AL for cross-border transactions). Both licenses require:

- Active CE in each state
- Each state's renewal fees
- Compliance with both states' advertising rules
- Affiliation with brokers in each state

This is common in metro areas spanning state lines.

## Reciprocity FAQ

- **Q: Does my CE from home state count for GA renewal?** Sometimes — usually not. Take GA-approved CE.
- **Q: Can I work full-time in GA on home-state license?** No — Georgia license required for active practice in GA.
- **Q: What if I move to GA permanently?** Apply for Georgia license through reciprocity; let home-state license go inactive or surrender it.
