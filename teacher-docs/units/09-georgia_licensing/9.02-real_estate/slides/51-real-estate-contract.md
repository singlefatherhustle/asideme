# Real Estate Sales Contract

## What it is

A legally binding agreement between buyer and seller for the transfer of real estate. In Georgia, this is almost always done on a standardized form (Georgia REALTORS Standard Form RE-1 or similar).

## Essential elements (the 6 + 1)

For a real estate contract to be enforceable in Georgia:

```
1. Offer  — proposal to buy at specific terms
2. Acceptance — seller's affirmative response
3. Consideration — earnest money or promise of payment
4. Competent parties — legal age, mental capacity
5. Legal purpose — sale is lawful
6. Mutual assent — both parties willingly agreed
+ Statute of Frauds — must be in writing
```

Missing any of these makes the contract unenforceable (you can't sue to enforce it).

## Standard form elements

Most real estate contracts cover:

| Section | What it specifies |
|---|---|
| Identification | Names of buyer, seller, broker |
| Property description | Address, parcel number, legal description |
| Purchase price | The agreed amount |
| Financing terms | Loan amount, type, contingency |
| Earnest money | Amount, when paid, when refundable |
| Inspection contingency | Right to inspect, deadlines |
| Appraisal contingency | Must appraise for purchase price |
| Closing date | Target date for closing |
| Possession date | When buyer takes possession |
| Title insurance | Required, who pays |
| Property condition | "As-is" or with warranties |
| Default | Remedies if either party breaches |
| Termination | When and how |
| Signatures | All parties + dates |

## Common contingencies

### Financing contingency
```
Buyer's obligation is conditional on obtaining mortgage approval.
If financing falls through, buyer can terminate without losing earnest money.
Deadline typically 30-45 days after contract.
```

### Inspection contingency
```
Buyer has X days (commonly 7-14) to inspect property.
If buyer finds significant issues, can:
   - Request repairs from seller
   - Terminate contract (with earnest money refund)
   - Negotiate price reduction
```

### Appraisal contingency
```
Property must appraise for purchase price.
If appraisal comes in low, buyer can:
   - Pay difference out of pocket
   - Negotiate price reduction
   - Terminate (if appraisal contingency invoked)
```

### Title contingency
```
Title search must show clear, marketable title.
If title issues exist, seller has time to cure.
If uncurable, buyer can terminate.
```

### HOA contingency (if applicable)
```
Buyer has time to review HOA documents.
If significant restrictions or disputes are revealed,
buyer can terminate within disclosure window.
```

## Performance and remedies

If either party breaches:

**Buyer's remedies if seller breaches**:
- Specific performance (force sale)
- Damages (lost down payment, alternative housing costs)
- Return of earnest money
- Attorney's fees per contract

**Seller's remedies if buyer breaches**:
- Specific performance (force purchase) — rarely chosen
- Damages
- Forfeiture of earnest money (most common)
- Attorney's fees per contract

## Time is of the essence

Most Georgia real estate contracts state "time is of the essence." This means:

- Every deadline matters
- Missing a deadline can void the contract
- Extensions must be in writing
- Excuses (illness, etc.) don't automatically extend

## Common contract errors

- Vague or missing property description
- Earnest money amount not specified
- Contingency deadlines not clearly stated
- "Subject to financing" without specifying loan type or amount
- Forgetting to initial each page or each substantive change
- Verbal modifications not converted to written addenda
- Closing date in the past or unreachable
