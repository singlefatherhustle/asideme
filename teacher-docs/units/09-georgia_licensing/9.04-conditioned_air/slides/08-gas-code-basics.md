# Georgia Fuel Gas Code Basics

## What it is

The Georgia Fuel Gas Code adopts the 2018 IFGC (International Fuel Gas Code) with Georgia amendments. Covers:

- Natural gas piping systems
- LP gas (propane) systems
- Gas appliance installation
- Venting
- Combustion air

## Pipe sizing — Section 402

Pipe size depends on:

```
1. Total connected appliance demand (BTU/hr)
2. Pipe length to each appliance (longest run = controlling)
3. Pressure drop allowance
4. Pipe material (Black iron, CSST, PE, etc.)
```

Common pipe sizing for natural gas (longest length method):

```
Branch to range (80,000 BTU/hr) at 50 ft from meter:
   Look up 80k at 50 ft in Table 402.4(2)
   Required: 1" black iron OR
            5/8" CSST (corrugated stainless steel tubing)
```

## Connection types

| Material | Use |
|---|---|
| Black iron pipe | Traditional, threaded connections |
| Galvanized steel | Outdoor / damp locations (rare) |
| CSST (Corrugated SS Tubing) | Flexible, fast install |
| Polyethylene (PE) | Underground only, transitions at ground level |
| Copper | Limited use (interior, accessible) |

## CSST grounding — Section 402.5

CSST requires special bonding to reduce lightning damage:

```
Bonding jumper:
   - Min #6 AWG copper
   - From CSST manifold to building grounding electrode system
   - Continuous, mechanically secured
```

This requirement is widely missed and is a common inspector callout.

## Appliance shut-off valves — Section 409

Each gas appliance must have:

- An accessible shut-off valve in the same room as the appliance
- Within 6 feet of the appliance for most cases
- A union or flange between valve and appliance for service

## Venting — Section 503

Gas appliance vents are categorized:

| Category | Description |
|---|---|
| **I** | Non-condensing, draft hood, natural vent |
| **II** | Non-condensing, fan-assisted, natural vent (rare) |
| **III** | Non-condensing, positive pressure |
| **IV** | Condensing, positive pressure |

Most residential furnaces today are Category IV (condensing, 90%+ efficient). They vent through PVC pipe.

## Direct-vent vs natural-vent

```
Natural vent: combustion air from inside, exhaust through chimney
Direct vent: combustion air from outside, exhaust through dedicated pipe
```

Direct-vent is required when:

- Appliance is in a small confined space
- Air-tight construction prevents drawing combustion air from inside
- High-efficiency Category IV equipment

## Combustion air — Section 304

Same general rule as IMC Chapter 7:

```
50 ft³ per 1,000 BTU/hr input
   OR provide outdoor air via screened openings (size per table)
```

Outdoor air openings:

- Upper opening near the ceiling
- Lower opening near the floor
- Size: 1 sq inch per 4,000 BTU/hr each opening

## Leak testing — Section 406

Pressure test before placing in service:

- Test pressure: 1.5 × working pressure, or 3 psig minimum for systems ≤ 1 psig
- Hold for 10 minutes minimum
- No measurable pressure drop allowed

## Common gas code violations

- Improper CSST bonding (major callout)
- Pipe size too small for connected load
- Missing appliance shut-off valve
- Improperly vented Category IV equipment (PVC mistakes)
- Combustion air openings undersized or missing
- Failed pressure test not retested before service
