# Georgia Mechanical Code Basics

## What it is

The Georgia Mechanical Code is the 2018 IMC (International Mechanical Code) with Georgia-specific amendments. It governs:

- HVAC equipment installation
- Ventilation requirements
- Combustion air for fuel-burning equipment
- Ductwork systems
- Refrigeration systems
- Hydronic systems

## Ventilation — Section 401

Outdoor air supply requirements:

```
Mechanical ventilation required when buildings are not natural ventilation alternative
Outdoor air rate calculated per:
   - Occupancy type
   - Floor area
   - Number of occupants
```

Common rates from Table 403.3:

| Occupancy | CFM/person + CFM/ft² |
|---|---|
| Office | 5 CFM/person + 0.06 CFM/ft² |
| Restaurant (dining) | 7.5 CFM/person + 0.18 CFM/ft² |
| Classroom | 10 CFM/person + 0.12 CFM/ft² |
| Retail | 7.5 CFM/person + 0.06 CFM/ft² |
| Industrial | varies by process |

## Combustion air — Section 701

For fuel-burning appliances:

```
50 ft³ of room volume per 1,000 BTU/hr of input rating
   OR provide outdoor air per code requirements
```

Example: 80,000 BTU/hr furnace requires:
- Room volume ≥ 80 × 50 = 4,000 ft³
- Standard 8-ft ceiling: 4,000 / 8 = 500 sq ft minimum room area
- If smaller: provide outdoor combustion air openings

## Chimneys and vents — Section 803

Vent connector sizing depends on:

- Appliance input rating
- Vent height
- Lateral run length

Use Table 803.2 — venting tables. Inadequate venting causes:

- Carbon monoxide hazards
- Combustion product backflow
- Equipment damage

## Refrigeration — Chapter 11

Industrial refrigeration:

- Specific safety requirements for large systems
- Refrigerant type classifications (A1, A2, A2L, A3, B1, etc.)
- Mechanical room requirements based on refrigerant quantity
- Pressure relief device specifications

## Inspector callouts

Common code violations:

- Inadequate combustion air for gas furnace
- Improperly sized exhaust hoods in commercial kitchens
- Missing makeup air for kitchen exhaust
- Inadequate vent connector sizing
- Mixed metal types in refrigeration piping (Cu-Al galvanic corrosion)

## What changes 2026

When Georgia adopts 2024 IMC:

- Updated ventilation rates (closer to ASHRAE 62.1 standards)
- New refrigeration requirements
- Updated controls for energy efficiency
- Carbon monoxide detection in new construction
