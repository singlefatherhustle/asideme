# Conditioned Air License Classes

## Two classes

Georgia divides conditioned air (HVAC) contractors into two classes:

| Class | Scope cap |
|---|---|
| **Class I** | Single-family dwellings + light commercial up to 175,000 BTU/hr cooling and 175,000 BTU/hr heating |
| **Class II** | Unrestricted — any size residential or commercial |

## What "175,000 BTU/hr" means

```
175,000 BTU/hr cooling ≈ 14.5 tons of cooling

Typical residential single-family AC: 24,000-60,000 BTU/hr (2-5 tons)
Typical light commercial: 60,000-200,000 BTU/hr
```

Class I covers most residential and small commercial work. Class II is required for larger commercial buildings.

## Why split this way

Georgia recognizes that residential HVAC and large commercial systems require different competencies:

- **Residential**: standard duct systems, single-stage equipment, common refrigerants
- **Commercial**: rooftop units, VRF, chillers, large duct networks, dedicated outdoor air, building automation, multiple-stage capacity controls

The 175k BTU/hr threshold roughly corresponds to where commercial-grade equipment typically starts.

## Class I scope examples

```
✓ Single-family home HVAC system
✓ Small office (≤175k BTU/hr cooling)
✓ Retail store with small system
✓ Multifamily up to 4 units (each unit individually metered)
```

## Class II scope examples

```
✓ Office building (5+ ton AC)
✓ Mid-size retail (10+ ton)
✓ Restaurant (commercial kitchens)
✓ Industrial process cooling
✓ Multifamily mid-rise with central plant
✓ Healthcare facility HVAC
```

## Experience and exam

| Requirement | Class I | Class II |
|---|---|---|
| Age | 21 | 21 |
| Experience | 2 years | 4 years (with commercial focus) |
| References | 3 (1 from licensed contractor) | 3 (1 from licensed contractor) |
| Exam | Yes | Yes |

## Application timing

The HVAC contractor application is straightforward:

```
1. Document experience (2 years Class I or 4 years Class II)
2. Get 3 references including a licensed contractor
3. Pay application + exam fees
4. Sit and pass exam
5. Initial license issued
```

## Common scope errors

- Class I contractor accepting a job over 175k BTU/hr cooling
- Class I contractor doing commercial new construction "because the building looks small"
- Class II credentials used for residential without recognizing operational differences (commercial billing rates apply at residential, customers complain)
