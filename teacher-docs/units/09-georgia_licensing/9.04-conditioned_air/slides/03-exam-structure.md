# HVAC Exam Structure

## The exam

Conducted by PSI Services at testing centers throughout Georgia.

| Attribute | Class I | Class II |
|---|---|---|
| Total questions | 80 | 90 |
| Time allotted | 200 min (3h 20m) | 230 min (3h 50m) |
| Passing score | 70 | 70 |
| Open-book | Yes | Yes |
| Calculator | Approved scientific | Approved scientific |

## Topic distribution (approximate)

```
General trade knowledge:        ~10%
Manual J (load calc):           ~15%
Manual D (duct design):         ~12%
Manual S (equipment selection): ~8%
Refrigerant handling:           ~10%
Georgia Mechanical Code:        ~12%
Georgia Fuel Gas Code:          ~8%
Georgia Energy Code:            ~8%
OSHA safety:                    ~7%
Business and law:               ~10%
```

## Approved references

Per the 2025 reference list:

- 2018 Georgia Mechanical Code (IMC + GA amendments)
- 2018 Georgia Fuel Gas Code
- ACCA Manual J (Residential Load Calc)
- ACCA Manual D (Duct Design)
- ACCA Manual S (Equipment Selection)
- 2018 Georgia Energy Code (IECC)
- EPA 608 Universal Certification material
- OSHA 29 CFR Part 1926
- NASCLA Business and Law for the business portion

## What changes Jan 1, 2026

Georgia adopts 2024 ICC family codes effective January 1, 2026, including:

- 2024 IMC (Mechanical Code)
- 2024 IFGC (Fuel Gas Code)
- 2024 IECC (Energy Code)

Exam reference lists update accordingly.

## Day-of routine

- Arrive 30 minutes early
- Two forms of ID required
- Approved references (clean copies, no handwritten notes)
- Calculator (no graphing or text-entry calculators)
- Personal items locked away

## Reference tabbing

Most candidates tab:

- Manual J load tables (most-referenced)
- Manual D duct sizing tables
- Manual S equipment selection criteria
- Mechanical code section 401 (ventilation)
- Fuel gas code section 504 (combustion air)

A well-tabbed reference saves time on each lookup. For a 200-minute exam with 80 questions, you have ~2.5 minutes per question.

## Common exam errors

- Bringing references with handwritten notes (auto-fail)
- Calculator with text capability (proctor confiscates)
- Pinning hopes on memorization vs reference (the exam expects you to use references)
- Spending too long on Manual J questions when the answer is in a table
- Skipping the business portion because it "seems easy"
