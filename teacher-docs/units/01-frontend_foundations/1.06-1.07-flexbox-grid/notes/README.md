# Flexbox & CSS Grid

## Flexbox - organize a site based on containers

- a container can only style it's immediate children
- in order to make a container, you must use { display: flex }

## CSS Grid - organize a site based on a system with rows and columns

- a container can only style it's immediate children
- in order to make a container, you must use { display: grid; }

## Grid tracks

- grid-template-columns - defines how WIDE each COLUMN will be
- grid-template-rows - defines how TALL each ROW will be
- grid-area-template + grid-area - allows for planning more complex elements
- repeat(# of times, width/height) - removes repetition

## Other Grid Properties

- grid-column-start / grid-column-end
- grid-row-start / grid-row-end
- column-gap / row-gap


## Media Queries

- apply CSS based on a condition
- requires 2 speciifc meta tags to work on mobile

## CSS Units

- px - pixels
- pt - points
- em - relative to the font-size of the element
- rem - relative to the font-size of the root element
- vw - relative to 1% of the width of the viewport
- vh - relative to 1% of the height of the viewport
- % relative to the parent element
