# Deploy a site on Netlify

We will be using [Netlify](https://www.netlify.com/) to host our websites.

1. Make an account if you don't already have one.
2. In the **Projects** tab of the Netlify dashboard, click the **Add new project**
   button and select **Deploy manually**.
3. Upload the _folder_ containing your `index.html` file of the website that
   you want to deploy.
4. Your project is now deployed! Netlify will give you a link that you can share.
   The format of this link will resemble `adjective-noun-12345.netlify.app`.

# Update a deployed site

What if you make changes and want to redeploy the newer version?

1. In the **Projects** tab of the Netlify dashboard, select the project you want to
   update.
2. Go to the **Deploys** tab of the project dashboard.
3. In the **Deploys** section, upload the folder containing the updated files.
4. Your project is now updated! The same link will now take users to the newer version
   of your website.
